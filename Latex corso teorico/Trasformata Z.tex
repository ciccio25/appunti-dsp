La trasformata-Z è la trasformata di Laplace nel dominio digitale, quindi dove il tempo non è più continuo, bensì è discreto. \newline 

\begin{tcolorbox}
    Essendo la trasformata-Z la trasformata di Laplace nel dominio digitale, 
    la si può utilizzare per studiare il transitorio della sequenza, come avevamo fatto in Laplace ad Elettronica, 
    ma, nel corso di DSP, la trasformata-z ci servirà solo per rappresentare la sequenza nel dominio z, senza studiare il transitorio. \newline  

    In Elettrotecnica avevamo studiato la trasformata di Laplace, in cui, data una funzione f(t) possiamo definire la sua trasformata di Lapace se esiste questo limite: 

    {
        \Large 
        \begin{equation}
            F(s) 
            = 
            \lim_{t \to +\infty} 
            \int_{0}^{t}
            f(\tau) \cdot e^{-s \tau}
            d\tau
        \end{equation}
    }

    dove: 

    {
        \Large 
        \begin{equation}
            s = \sigma + \jmath \omega
        \end{equation}
    }

    In particolare, noi abbiamo studiato nel corso di Elettrotecnica la trasformata di Laplace unilatera, 
    cioè in cui l'integrale della trasformata parte dal tempo $t = 0$ e t che tende a $+ \infty$, proprio perchè i sistemi reali (condensatori, induttori, resistori, ecc...) 
    sono componenti causali. \newline 
    
    Invece, nella forma più generale, possiamo definire la trasformata di Laplace bilatera come: 

    {
        \Large 
        \begin{equation}
            F(s) 
            = 
            \int_{- \infty}^{+\infty}
            f(\tau) \cdot e^{-s \tau}
            d\tau
        \end{equation}
    }

    e quindi dove l'integrale viene svolto nell'intervallo $\left[ - \infty, + \infty\right]$. \newline 

    Utilizzando la trasformata di Laplace bilatera, si possono studiare anche sistema non causali. \newline 

    Se vuoi un piccolo ripasso della trasformata di Laplace: 

    \begin{itemize}
        \item Trasformata di Laplace - Wikipedia\\ \url{https://it.wikipedia.org/wiki/Trasformata_di_Laplace} 
        \item But what is a Laplace Transform? by 3Blue1Brown\\ \url{https://youtu.be/j0wJBEZdwLs?si=ZzA0q2zRbt_2Nfrc}
    \end{itemize}
\end{tcolorbox}

\newpage 

Come nella trasformata di Laplace, possiamo definire due tipi di trasformate-Z. \newline 

La trasformata-z bilatera ha la seguente forma: 

{
    \Large 
    \begin{equation}
        X \left(z\right)
        = 
        Z \left\{ 
            x \left[ n\right]
        \right\}
        = 
        \sum_{n = -\infty}^{+ \infty}
        x \left[ n\right]
        \cdot 
        z^{-n}
    \end{equation}
}

dove $x \left[ n\right]$ è una sequenza a tempo discreto. \newline 

Notiamo che la trasformata-z è bilatera perchè la sommatoria è svolta per $n \in \left[ - \infty, + \infty\right]$. \newline 

\begin{tcolorbox}
    La z in $X\left(z\right)$ non è una variabile discreta, bensì è continua perchè ci troviamo nel piano complesso. 
\end{tcolorbox}

Oltre alla trasformata-z bilatera, è possibile definire la trasformata-z monolatera come: 

{
    \Large 
    \begin{equation}
        X \left(z\right)
        = 
        Z \left\{ 
            x \left[ n\right]
        \right\}
        = 
        \sum_{n = 0}^{+ \infty}
        x \left[ n\right]
        \cdot 
        z^{-n}
    \end{equation}
}

Notiamo che la trasformata-z è unilatera perchè la sommatoria è svolta per n che parte da 0 e finisce a $+\infty$. \newline 

\begin{tcolorbox}
    La trasfomata-z unilatera è la trasformata che ci interessa di più a fini ingegneristici perchè, per la maggior parte dei casi, 
    avremo a che fare con segnali e sistemi causali "real time", e che quindi partono dall'indice $n = 0$. 
\end{tcolorbox}

Possiamo svolgere le seguenti osservazioni: 

\begin{itemize}
    \item il piano Z in cui si svolge la trasformata Z è un piano complesso (e quindi con un asse della parte reale ed immaginaria) 
    \item la trasformata-z bilatera e unilatera coincidono se le sequenze sono causali, cioè: 
    {
        \Large 
        \begin{equation}
     x \left[ n\right] = 0 \quad \forall n < 0        
        \end{equation}
    }
    
    \item la trasformata bilatera coincide con la DTFT per: 
    {
        \Large 
        \begin{equation}
     z = e^{\jmath \omega}       
        \end{equation}
    } 
    se questa esiste (vedi \ref{subsec: Esistenza della trasformata} \nameref{subsec: Esistenza della trasformata} a pag \pageref{subsec: Esistenza della trasformata}) 
    \item sia la trasformata-z bilatera che quella monolatera sono operatori lineari, cioè :
    
    {
        \Large 
        \begin{equation}
       Z \left\{a \cdot x_1 \left[ n\right] + b \cdot x_2 \left[ n\right]\right\} 
       = 
       a \cdot Z \left\{ x_1 \left[ n\right]\right\} + b \cdot Z \left\{ x_2 \left[ n\right]\right\}     
        \end{equation}
    } 
\end{itemize}

\subsection{Regione di convergenza - ROC}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 3\\
    Appunti| 2025-10-15 | pag 1 \\ 
    Latex | 2025-10-21 Mattino | pag 114 - 115
}

In generale la serie $X \left(z \right)$ converge \textbf{solo} per certi valori di z. \newline 

Se poniamo z come: 

{
    \Large 
    \begin{equation}
        z = \abs{z} \cdot e^{\jmath \omega}
    \end{equation}
} 

allora la trasformata-z bilatera sarà uguale a (facendo le sostituzioni): 

{
    \Large 
    \begin{equation}
        \begin{split}
        X \left(z\right)
        = 
        Z \left\{ 
            x \left[ n\right]
        \right\}
        &= 
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right]
        \cdot 
        z^{-n}
        \\
        &\downarrow
        \\
        X \left(z\right)
        &=
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right]
        \cdot 
        \left(
            \abs{z} \cdot e^{\jmath \omega}
        \right)^{-n}
        \\
        &=
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right]
        \cdot 
        \abs{z}^{-n} 
        \cdot 
        e^{-\jmath \omega n}
        \end{split}
    \end{equation}
}

La trasformata-z bilatera $X \left(z\right)$ converge se: 

{
    \Large 
    \begin{equation}
        \sum_{n = - \infty}^{+ \infty}
        \abs{
        x \left[ n\right]
        \cdot 
        \abs{z}^{-n}
        }
        < 
        + \infty
    \end{equation}
}

che è una condizione sufficiente. \newline 

Cioè, detta in parole, la sommatoria converge se è assolutamente sommabile (vedi il modulo per indicare il valore assoluto tra $ x \left[ n\right]\cdot \abs{z}^{-n}$). \newline 

\newpage 

Possiamo svolgere delle interessanti osservazioni in merito alla regione di convergenza (chiamata anche ROC, dall'inglese Region of Convergence) nella trasformata-z: 

\begin{itemize}
    \item la trasformata-z può essere applicata ad una classe di sequenze più ampie rispetto alla trasformata di Fourier per le sequenze a tempo discreto (cioè la DTFT)
    \item se la trasformata-z converge per un valore: 
    {
        \Large 
        \begin{equation}
            z_0 = \abs{z_0} \cdot e^{\jmath \omega_0}
        \end{equation}
    }
    allora la trasformata-z converge in tutti i punti della circonferenza di raggio $\abs{z_0}$ e con centro l'origine, come si vede bene in figura: 
    \begin{figure}[h]
    \centering
    \includegraphics[scale = 0.4]{convergenza per una ROC.png}
    \end{figure} 
    \item calcolare la trasformata di Fourier a tempo discreto (cioè la DTFT) equivale a valutare la trasformata Z sul cerchio unitario perchè il modulo di $e^{\jmath \omega_0}$ è proprio uguale a 1, che è il raggio del cerchio unitario della trasformata-z 
\end{itemize}

\subsubsection{Esempi di calcolo della trasformata z}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 4 - 5\\
    Appunti| 2025-10-15 | pag 2
}

Le seguenti formule sono assai utili: 

{
    \Large 
    \begin{equation}
        \sum_{n = 0}^{+ \infty}
        \alpha^{-n}
        = 
        \dfrac{1}{1 - \alpha^{-1}}
        \text{ per } \abs{\alpha} < 1
    \end{equation}
}

cioè una serie geometria da $n = 0$ a $n = + \infty$ tende a quel termine. \newline 

\begin{tcolorbox}
    La serie geometrica l'avevamo viste anche ad Analisi Matematica 1: \newline 

    Serie geometrica - Wikipedia \url{https://it.wikipedia.org/wiki/Serie_geometrica}
\end{tcolorbox}

Invece se è una geometrica finita, cioè per una sommatoria con $n \in \left[ n_1, n_2\right]$: 

{
    \Large 
    \begin{equation}
        \sum_{n = n_1}^{n_2}
        \alpha^{-n}
        = 
        \dfrac{\alpha^{- n_1} - \alpha^{- \left( n_1 + 1\right)} }{1 - \alpha^{-1}}
    \end{equation}
}

allora possiamo continuare con gli esercizi di esempio. \newline 

\begin{tcolorbox}
Tutti i calcoli che svolgo in queste pagine, ve li potete risparmiare, non sono necessari per l'esame: serve solo a capire come ci si arriva alle formule indicate nelle slides. \newline 

Considerate solo l'inizio e la fine di ogni esercizio.
\end{tcolorbox}

\textbf{Esercizio 1} \newline 

Se consideriamo la successione $x \left[ n\right]$: 

{
    \Large 
    \begin{equation}
        x \left[ n\right]
        = 
        \alpha^{n} \cdot u \left[ n\right]
    \end{equation}
} 

dove: 

\begin{itemize}
    \item $\alpha^{n}$ è un esponenziale 
    \item $u \left[ n\right]$ è un gradino
\end{itemize}

allora, possiamo definire la trasformata-z bilatera di $x \left[ n\right]$ come: 

{
    \Large 
    \begin{equation}
        X \left(z\right) 
        = 
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right] \cdot z^{- n} 
    \end{equation}
}

Siccome $u \left[ n\right]$ è un gradino, cioè una sequenza causale, 
allora possiamo definire la trasformata-z della sequenza $x \left[ n\right]$ come: 

{
    \Large 
    \begin{equation}
        \setlength{\jot}{10pt} % affecting the line spacing in the environment
        \begin{split}
        X \left(z\right) 
        &= 
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right] \cdot z^{- n}
        \\
        &\downarrow
        \\
        X \left(z\right) 
        &= 
        \sum_{n = 0}^{+ \infty}
        x \left[ n\right] \cdot z^{- n}     
        \end{split}
    \end{equation}
}

cioè, la trasformata-z da bilatera diventa unilatera a causa della sequenza causale e la sommatoria parte da $n = 0$ e non da $n = - \infty$. \newline 

Continuando con la sommatoria (step-by-step): 

{
    \Large 
    \begin{equation}
        \setlength{\jot}{10pt} % affecting the line spacing in the environment
        \begin{split}
        X \left(z\right) 
        &= 
        \sum_{n = 0}^{+ \infty}
        x \left[ n\right] \cdot z^{- n}     
        \\
        &= 
        \sum_{n = 0}^{+ \infty}
        \alpha^{n} \cdot z^{- n} 
        \\
        &= 
        \sum_{n = 0}^{+ \infty}
        \left( \alpha \cdot z^{-1}\right)^{n}
        \\
        &= 
        \dfrac{1}{1 - \alpha \cdot z^{-1}}
        \end{split}
    \end{equation}
}

$\dfrac{1}{1 - \alpha \cdot z^{-1}}$ è una funzione razionale reale. \newline 

\begin{tcolorbox}
    Un piccolo ripasso al volo di cosa è una funzione razionale reale: \newline 

    Funzione razione - Wikipedia \url{https://it.wikipedia.org/wiki/Funzione_razionale}
\end{tcolorbox}

Siccome la radice del polinomio è uguale ad $\alpha$, 
allora $X \left(z\right)$ sarà discontinua in quel punto. \newline 

\begin{tcolorbox}
    Un ripasso di cosa si intende per radice di un polinomio: \newline 
    
    \url{https://www.andreaminini.org/matematica/algebra/zeri-di-un-polinomio}
\end{tcolorbox}

Quindi, $X \left( z\right)$ converge se: 

{
    \Large 
    \begin{equation}
        \abs{z} > \abs{\alpha}
    \end{equation}
}

Nel piano z, possiamo vedere $X \left(z\right)$ in questa maniera: 

\begin{figure}[h]
\centering
\includegraphics[scale = 0.5]{Esercizio 1 calcolo della convergenza trasformata z.png}
\end{figure} 

dove in tratteggiato sono tutti i punti nel piano dove può convergere $X \left(z\right)$ se $\abs{z} > \abs{\alpha}$. \newline 

\textbf{Esercizio 2} \newline 

Data la seguente sequenza: 

{
    \Large 
    \begin{equation}
        x \left[ n\right] = - \alpha^{n} \cdot u \left[ - n - 1\right]
    \end{equation}
}

\begin{itemize}
    \item $\alpha^{n}$ è un esponenziale 
    \item $u \left[ - n - 1\right]$ è un gradino ribaltato, shiftato di -1 quindi non causale perchè per $n = 0$ avremo $u \left[- 1\right]$ cioè un valore precedente rispetto all'istante iniziale di osservazione, che appunto è $n = 0$ 
\end{itemize}

Allora possiamo formulare la trasformata-z bilatera di $x \left[ n\right]$ come: 

{
    \Large 
    \begin{equation}
        \begin{split}
            X \left( z\right)
            = 
            \sum_{n = -\infty}^{+ \infty}
            x \left[ n\right] \cdot z^{-n}
        \end{split}
    \end{equation}
}

\newpage 

Siccome la sequenza $x \left[ n\right]$ si estende solo da $n \in \left[ - \infty, - 1\right]$, 
allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        \begin{split}
            X \left( z\right)
            &= 
            \sum_{n = -\infty}^{+ \infty}
            x \left[ n\right] \cdot z^{-n}
            \\
            &\downarrow
            \\
            X \left( z\right)
            &= 
            \sum_{n = -\infty}^{- 1}
            \left( - \alpha^{n}\right) \cdot z^{-n}
            \\
            &= 
            -
            \sum_{n = 1}^{+ \infty}
            \left(  \dfrac{z}{\alpha}\right)^{n}
            \\
            &= 
            -
            \left(  \dfrac{z}{\alpha}\right)
            \cdot
            \sum_{n = 0}^{+ \infty}
            \left(  \dfrac{z}{\alpha}\right)^{n}
            \\
            &= 
            \dfrac{- \left(  \dfrac{z}{\alpha}\right) \cdot 1}{1 - \left(  \dfrac{z}{\alpha}\right)}
            \\
            &= 
            \dfrac{1}{1 - \alpha \cdot z^{-1}}
        \end{split}
    \end{equation}
}

Cioè la serie è come l'esercizio 1, ma in questo caso converge se: 

{
    \Large 
    \begin{equation}
        \abs{z} < \abs{\alpha}
    \end{equation}
}

Nel piano z, possiamo vedere $X \left(z\right)$ in questa maniera: 

\begin{figure}[h]
\centering
\includegraphics[scale = 0.5]{Esercizio 2 calcolo della convergenza trasformata z.png}
\end{figure} 

dove, in figura, in tratteggiato è la stessa regione di convergenza dell'esercizio 1 ma in senso opposto. \newline

\section{Alcune trasformate-z notevoli} \label{sec: Alcune trasformate-z notevoli}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 6 \\
    Appunti| 2025-10-15 | pag 2
}

Come nelle altre trasformate, come la DTFT, anche nella trasformata-z ci sono delle trasformate notevoli (cioè che ogni volta non dobbiamo calcolarci). \newline 

Di seguito le principali, dove a sinistra indicheremo nel dominio delle sequenze n, invece a destra nel dominio z delle trasformate-z. \newline 

La trasformata-z di un impulso $\delta \left[ n\right]$ nel dominio n, è uguale a 1 nel dominio z, cioè: 

{
    \Large 
    \begin{equation}
        \delta \left[ n\right]
        \leftrightarrow 
        1
    \end{equation}
}
 
in cui la regione di convergenza (cioè la ROC) è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ 1\right\} = \forall z
    \end{equation}
}

La trasformata-z di un gradino $u \left[ n\right]$ nel dominio n, è uguale alla seguente funzione razionale nel dominio z: 

{
    \Large 
    \begin{equation}
        u \left[ n\right] 
        = 
        \dfrac{1}{1 - z^{-1}}
    \end{equation}
}

dove la radice del polinomio è uguale a 1, quindi la sua ROC è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ \dfrac{1}{1 - z^{-1}} \right\} = \abs{z} > 1
    \end{equation}
}

La trasformata-z di un gradino ribaltato $- u \left[ - n - 1\right]$ nel dominio n, è uguale alla seguente funzione razionale nel dominio z: 

{
    \Large 
    \begin{equation}
        - u \left[ - n - 1\right] 
        = 
        \dfrac{1}{1 - z^{-1}}
    \end{equation}
}

dove la radice del polinomio è uguale a 1, quindi la sua ROC è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ \dfrac{1}{1 - z^{-1}} \right\} = \abs{z} < 1
    \end{equation}
}

cioè è la stessa del gradino $u \left[ n\right]$ ma in verso opposto. \newline 

La trasformata-z di una retta moltiplicata per un esponenziale moltiplicato a sua volta per un gradino, 
cioè $n \cdot \alpha^{n} \cdot  u \left[ n\right]$ nel dominio n, è uguale alla seguente funzione razionale nel dominio z: 

{
    \Large 
    \begin{equation}
        n \cdot \alpha^{n} \cdot  u \left[ n\right]
        = 
        \dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}}
    \end{equation}
}

dove la radice del polinomio è uguale a $\alpha$, quindi la sua ROC è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ \dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}} \right\} 
        = 
        \abs{z} > \abs{\alpha}
    \end{equation}
}

\newpage 

La trasformata-z di una retta negativa moltiplicata per un esponenziale moltiplicato a sua volta per un gradino ribaltato e ritardato di 1, 
cioè $- n \cdot \alpha^{n} \cdot  u \left[ -n -1 \right]$ nel dominio n, è uguale alla seguente funzione razionale nel dominio z: 

{
    \Large 
    \begin{equation}
        -n \cdot \alpha^{n} \cdot  u \left[ -n - 1\right]
        = 
        \dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}}
    \end{equation}
}

dove la radice del polinomio è uguale a $\alpha$, quindi la sua ROC è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ \dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}} \right\} 
        = 
        \abs{z} < \abs{\alpha}
    \end{equation}
}


cioè è la stessa del gradino $n \cdot \alpha^{n} \cdot  u \left[ n\right]$ ma in verso opposto. \newline 

Dopo l'elenco di alcune trasformate notevoli, possiamo svolgere alcune osservazioni. \newline 

Per avere corrispondenza biunivoca tra sequenze e trasformata occorre specificare la regione di convergenza (cioè la ROC). \newline 

Prendendo per esempio la funzione razionale $\dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}}$, 
se indichiamo come ROC $\abs{z} < \abs{\alpha}$, avremo, nel dominio n, la sequenza $-n \cdot \alpha^{n} \cdot  u \left[ -n - 1\right]$. \newline 

Se abbiamo la stessa funzione razionale $\dfrac{\alpha z^{-1}}{\left( 1 - \alpha z^{-1}\right)^{2}}$, 
ma in questo caso indichiamo come ROC $\abs{z} > \abs{\alpha}$, avremo, nel dominio n, la sequenza $n \cdot \alpha^{n} \cdot  u \left[ n\right]$. \newline 

Inoltre, un'altra osservazione è che le trasformate di nostro interesse in questo corso di DSP sono, in genere, 
funzioni razionali della variabile z o della variabile $z^{-1}$. \newline

\subsection{Trasformazione inversa}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 10 \\
    Appunti| 2025-10-15 | pag 2
}

Come tutte le trasformate, una volta che si sono svolti i conti in un dominio, in questo caso quello delle z, 
si vuole ritornare nel dominio delle sequenze n. \newline 

Come specificato nelle osservazioni riguardo le trasformate notevoli, 
specificando la ROC, c'è corrispondenza biunivoca tra $x \left[ n\right]$ nel dominio n e la sua trasformata-z $X \left( z\right)$. \newline 

Esistono principalmente tre metodi per svolgere la trasformazione inversa: 

\begin{itemize}
    \item metodo per ispezione 
    \item metodo per espansione in serie 
    \item espansione in fratti semplici
\end{itemize} 

Adesso specifichiamo come si svolgono i vari metodi. \newline 

Nel metodo per ispezione si utilizzano le tabelle sequenza-trasformata all'inverso 
(vedi \ref{sec: Alcune trasformate-z notevoli} \nameref{sec: Alcune trasformate-z notevoli} pag \pageref{sec: Alcune trasformate-z notevoli} ) \newline 

Ad esempio: 

{
    \Large 
    \begin{equation}
        X \left(z\right)
        =
        \dfrac{5}{\alpha \cdot z^{-1}} 
    \end{equation}
}

in cui la ROC è: 

{
    \Large 
    \begin{equation}
        \abs{z} > \abs{\alpha}
    \end{equation}
}

allora, sapendo la ROC e $X \left( z\right)$ possiamo ritornare nel dominio n delle sequenze, cioè: 

{
    \Large 
    \begin{equation}
        x \left[ n\right]
        = 
        5 \cdot \alpha^{n} \cdot u \left[ n\right]
    \end{equation}
}

Nel metodo per espansione in serie, invece, ci si riporta ad un'espressione come serie di potenze in $z^{-1}$, 
cioè i coefficienti sono i valori delle sequenze \newline 

Un esempio pratico: 

\begin{figure}[h]
\centering
\includegraphics[scale = 0.5]{Metodo per espansione in serie trasformata z.png}
\end{figure} 

\newpage 

Invece, nel metodo ad espansione in fratti semplici, se $X \left(z\right)$ è una funzione razionale, 
si può utilizzare lo stesso approccio utilizzato in Laplace, cioè quello di espandere $X \left(z\right)$ in fratti semplici e poi fare la trasformazione inversa dei singoli componenti con le tabelle 
(vedi \ref{sec: Alcune trasformate-z notevoli} \nameref{sec: Alcune trasformate-z notevoli} pag \pageref{sec: Alcune trasformate-z notevoli} ). \newline

\section{Proprietà della ROC} \label{sec: Proprietà della ROC}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 7  - 8\\
    Appunti| 2025-10-15 | pag 2 \\ 
    Latex | 2025-10-21 Mattino | pag 124
} 

Come detto precedentemente, nella ROC la trasformata è una funzione analitica (cioè è una funzione continua tranne nelle discontinuità fuori o al bordo della ROC). \newline 

Per sequenze di durata finita come la seguente: 

{
    \Large 
    \begin{equation}
        \sum_{n = n_1}^{n_2}
        x \left[ n\right] \cdot z^{-n}
    \end{equation}
}

dove: 

{
    \Large 
    \begin{equation}
        \abs{n_1}, \abs{n_2} < + \infty
    \end{equation}
}

la ROC è la seguente: 

{
    \Large 
    \begin{equation}
        ROC = \forall z 
    \end{equation}
}

in particolare da verificare per i punti $z = 0$ e $z = +\infty$. \newline 

\begin{tcolorbox}
    Ingegneristicamente parlando, non ci interessa verificare la ROC per i punti $z = 0$ e $z = +\infty$ : è solo un avvezzo matematico. 
\end{tcolorbox}

Per sequenze monolatere destre come la seguente:

{
    \Large 
    \begin{equation}
        \sum_{n = n_1}^{+ \infty}
        x \left[ n\right] \cdot z^{-n}
    \end{equation}
}

dove: 

{
    \Large 
    \begin{equation}
        n_1 > - \infty
    \end{equation}
}

la ROC è la seguente: 

{
    \Large 
    \begin{equation}
        ROC = \forall z: \abs{z} > \abs{z_0}
    \end{equation}
}

\begin{tcolorbox}
Dal punto di vista ingegneristico, la maggior parte delle volte avremo a che fare con questo tipo di sequenze, 
cioè le sequenze monolatere destre perchè sono sequenze causali, a differenza delle sequenze monolatere sinistre, 
dove le sequenze non sono causali, chiamate anche sequenze "non real-time", perchè l'elaborazione "è all'indietro" nel tempo.     
\end{tcolorbox}

\newpage 

Per sequenze monolatere sinistre come la seguente:

{
    \Large 
    \begin{equation}
        \sum_{n = - \infty}^{n_2}
        x \left[ n\right] \cdot z^{-n}
    \end{equation}
}

dove: 

{
    \Large 
    \begin{equation}
        n_2 < + \infty
    \end{equation}
}

la ROC è la seguente: 

{
    \Large 
    \begin{equation}
        ROC = \forall z: \abs{z} < \abs{z_0}
    \end{equation}
}

Invece per sequenze (bilatere) generiche, se esiste una ROC avremo che: 

{
    \Large 
    \begin{equation}
        ROC = \forall z: \abs{z_1}< \abs{z} < \abs{z_2}
    \end{equation}
}

cioè la ROC è una regione anulare. \newline

Considerando una funzione razionale in z: 

{
    \Large 
    \begin{equation}
        X \left( z\right)
        = 
        \dfrac{N \left(z\right)}{D \left(z\right)}
    \end{equation}
}

dove: 

\begin{itemize}
    \item $N \left(z\right)$ è il polinomio z numeratore di $X \left( z\right)$ 
    \item $D \left(z\right)$ è il polinomio z denominatore di $X \left( z\right)$ 
\end{itemize}

Spesso si preferisce la forma (equivalente) in $z^{-1}$ perchè è più comodo. \newline 

Si chiamano zeri di $X \left(z\right)$ tutti quei valori per cui $X \left( z\right) \to 0$. \newline 

Si chiamano poli di $X \left(z\right)$ tutti quei valori per cui $X \left( z\right) \to + \infty$. \newline 

I poli sono: 

\begin{itemize}
    \item le radici del polinomio $D \left( z\right)$ 
    \item se necessario anche $z = \infty$ è un polo
\end{itemize}

I poli solo le uniche discontinuità della funzione razionale $X \left(z\right)$. \newline

\section{Proprietà della trasformata-z} \label{sec: Proprietà della trasformata-z}

\footnote{ 
    Slide del prof | 5 - Trasformata Z | pag 11 - 13\\
    Appunti| 2025-10-15 | pag 3 \\ 
    Latex | 2025-10-21 Mattino | pag 129 - 130
} 

\begin{tcolorbox}
    Le proprietà della trasformata-z sono simili a quelle della trasformata di Laplace. \newline 

    Le proprietà della trasformata-z non verranno richieste all'esame, ma saranno molto utili dal punto di vista pratico e per capire la relazione con la trasformata di Laplace.
\end{tcolorbox}

\begin{comment}
\begin{tcolorbox}[colback=green!5!white,colframe=green!75!black]

Come ogni trasformata, anche la trasformata-z rispetta delle proprietà. \newline 

\textbf{Linearità} \newline 

Date: 

\begin{itemize}
    \item due sequenze nel dominio n $x_1$ e $x_2$ e le rispettive trasformate-z, quindi $X_1 \left(z\right)$ e $X_2 \left(z\right)$ 
    \item due coefficienti a e b 
\end{itemize}

allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        a \cdot x_1 \left[ n\right]
        + 
        b \cdot x_2 \left[ n\right]
        \leftrightarrow 
        a \cdot X_1 \left( z\right)
        + 
        b \cdot X_2 \left( z\right)
    \end{equation}
}

in cui la ROC dell'unione tra i due è: 

{
    \Large 
    \begin{equation}
        ROC \left\{ 
        a \cdot X_1 \left( z\right)
        + 
        b \cdot X_2 \left( z\right)
        \right\}
        = 
        ROC \left( X_1\right) 
        \cap
        ROC \left( X_1\right) \text{ come minimo} 
    \end{equation}
}

\textbf{Traslazione nel tempo} \newline 

Data: 

\begin{itemize}
    \item una sequenza nel dominio n $x \left[ n\right]$ e la sua trasformata-z $X \left(z\right)$
    \item un coefficiente di ritardo $n_0$ nel dominio n
\end{itemize}

allora possiamo esprimere che: 

{
    \Large 
    \begin{equation}
        x \left[ n - n_0\right] 
        \leftrightarrow
        z^{- n_0} \cdot X \left( z\right)
    \end{equation}
}

in cui la ROC vale: 

{
    \Large 
    \begin{equation}
        ROC \left\{ z^{- n_0} \cdot X \left( z\right) \right\}
        = 
        ROC \left( X \right) 
        \text{ eccetto per } z = 0 \text{ oppure } z = + \infty
    \end{equation}
} 

\newpage 

\textbf{Derivazione in Z} \newline 

Data una sequenza $x \left[ n\right]$ e la sua trasformata-z $X \left(z\right)$, 
allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        n \cdot x \left[ n\right]
        \leftrightarrow 
        - z \cdot \dfrac{\partial X \left( z\right)}{\partial z}
    \end{equation}
}

in cui la ROC vale: 

{
    \Large 
    \begin{equation}
        ROC \left\{ z^{- n_0} \cdot X \left( z\right) \right\}
        = 
        ROC \left( X \right) 
        \text{ eccetto per } z = 0 \text{ oppure } z = + \infty
    \end{equation}
} 

\begin{tcolorbox}
    Possiamo notare che la ROC, se si applica una traslazione nel tempo o una derivazione in z, 
    non cambia. 
\end{tcolorbox}

\textbf{Moltiplicazione per una sequenza esponenziale} \newline 

Data: 

\begin{itemize}
    \item una sequenza nel dominio n $x \left[ n\right]$ e la sua trasformata-z $X \left( z\right)$ 
    \item un coefficiente moltiplicativo del tipo $z_0$
\end{itemize}

possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        z_0^{n} \cdot x \left[ n\right]
        \leftrightarrow 
        X \left( \dfrac{z}{z_0}\right)
        \quad
        \text{ dove } z_0 \text{ costante}
    \end{equation}
}

in cui la ROC vale: 

{
    \Large 
    \begin{equation}
        ROC \left\{ X \left( \dfrac{z}{z_0}\right) \right\}
        = 
        \abs{z_0} \cdot ROC \left( X\right) \text{ scalato}
    \end{equation}
}

Se $d_k$ è un polo di $X \left(z\right)$, allora $z_0 d_k$ è un polo di $X \left( \dfrac{z}{z_0}\right)$. \newline 

Per questo motivo, possiamo scrivere le sequenti proprietà: 

\begin{itemize}
    \item Se $z_0 \in \mathbb{R}$ allora i poli e zeri di $X \left( z\right)$ in $X \left( \dfrac{z}{z_0}\right)$ si muovono radialmente 
    \item Se $z_0 = e^{\jmath \omega_0}$ allora i poli e zeri di $X \left( z\right)$ in $X \left( \dfrac{z}{z_0}\right)$ ruotano di $\omega_0$ ( vedi traslazione in frequenza della DTFT in \ref{subsubsec: Proprietà della trasformata di Fourier} \nameref{subsubsec: Proprietà della trasformata di Fourier} pag \pageref{subsubsec: Proprietà della trasformata di Fourier}) dove ruotano in senso orario se $\omega_0$ è positivo, in senso anti-orario se $\omega_0$ è negativo
    \item Se $z_0 \in \mathbb{C}$ allora si eseguono tutti e due gli effetti contemporaneamente in $X \left( \dfrac{z}{z_0}\right)$
\end{itemize}

\textbf{Coniugazione complessa} \newline 

Data una sequenza $x \left[ n\right]$ nel dominio n e la sua trasformata-z $X \left(z\right)$, 
allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        x^{*} \left[ n\right]
        \leftrightarrow 
        X^{*} \left(z^{*}\right)
    \end{equation}
}

dove: 

{
    \Large 
    \begin{equation}
        ROC \left\{ X^{*} \right\}
        = 
        ROC \left( X \right) 
    \end{equation}
} 

\newpage 

\textbf{Rovesciamento nel tempo} \newline

Data una sequenza nel dominio n $x \left[ n\right]$ e la sua trasformata-z $X \left( z\right)$, 
possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        x \left[ - n\right]
        \leftrightarrow 
        X \left( \dfrac{1}{z}\right)
    \end{equation}
}

dove la ROC vale: 

{
    \Large 
    \begin{equation}
        ROC \left\{ X \left( \dfrac{1}{z}\right) \right\} 
        = 
        \dfrac{1}{ROC \left(X\right)}
    \end{equation}
}

infatti una sequenza a destra, a causa del rovesciamento nel tempo, diventa sinistra e viceversa. \newline 

\textbf{Convoluzione di sequenze} \newline

\begin{tcolorbox}
    Questa è la proprietà più importante della trasformata-z 
\end{tcolorbox}

Date due sequenze nel dominio n $x \left[ n\right]$ e $y \left[ n\right]$ e le rispettive trasformate-z, 
cioè $X \left( z\right)$ e $Y \left( z\right)$, allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        x \left[ n\right] * y \left[ n\right]
        \leftrightarrow 
        X \left(z \right) \cdot Y \left( z\right)
    \end{equation}
}

cioè la convoluzione tra $x \left[ n\right]$ e $y \left[ n\right]$ nel dominio n è uguale al prodotto delle rispettive trasformate-z nel dominio z. \newline 

Inoltre: 

{
    \Large 
    \begin{equation}
        ROC \left\{ X \left(z \right) \cdot Y \left( z\right) \right\} 
        \text{ contiene }
        ROC \left(X\right) \cap ROC \left( Y\right)
    \end{equation}
}

\textbf{Valore iniziale e valore finale} \newline 

Data una sequenza $x \left[  n \right]$ nel dominio n causale dove: 

{
    \Large 
    \begin{equation}
        x \left[ n\right] 
        = 
        0 
        \quad
        \forall n < 0
    \end{equation}
}

e sapendo che la trasformata-z di $x \left[ n\right]$ è $X \left(z \right) $, 
allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        x \left[ 0\right]
        = 
        \lim_{z \to + \infty} X \left(z\right)
    \end{equation}
}

e 

{
    \Large 
    \begin{equation}
        x \left[ + \infty \right]
        = 
        \lim_{z \to 1^{+}} \left( z - 1\right) \cdot X \left(z\right)
    \end{equation}
}
