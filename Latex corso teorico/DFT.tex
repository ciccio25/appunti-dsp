\section{Sequenze periodiche}

\footnote{ 
    Slide del prof | 7 - DFT | pag 2 - 3\\
    Appunti| 2025-10-28 | pag 4   
}

Consideriamo una sequenza periodica $\tilde{x} \left[ n\right]$ per cui: 

{
    \Large 
    \begin{equation}
        \tilde{x} \left[ n\right] 
        = 
        \tilde{x} \left[ n + N\right] 
    \end{equation}
}

dove N è un numero intero. \newline 

\begin{tcolorbox}
    A livello di notazione, scriveremo $\tilde{x} \left[ n\right]$, cioè una sequenza $x \left[ n\right]$ periodica, 
    se è presente un'ondina \~ sopra alla sequenza. 
\end{tcolorbox}

Di una sequenza periodica $\tilde{x} \left[ n\right]$ possiamo determinare la sua pulsazione (fondamentale) $\omega_0$ come: 

{
    \Large 
    \begin{equation}
        \omega_0 = \dfrac{2 \pi}{N}
    \end{equation}
}

A titolo di esempio, possiamo utilizzare la seguente figuta: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.3]{Sequenza periodica di esempio.png}
\end{figure} 

Come si nota dalla figura, la conoscenza degli N campioni, cioè: 

{
    \Large 
    \begin{equation}
        \tilde{x} \left[ k\right] \text{ per } k = 0, 1, \dots, \left(N - 1\right)
    \end{equation}
}

permette di conoscere l'intera sequenza periodica. \newline 

Proprio perchè una sequenza periodica $\tilde{x} \left[ n\right]$ non è nè assoltamente sommabile, 
nè quadraticamente sommabile, allora, per calcolare la sua DTFT la calcoleremo come: 

{
    \Large 
    \begin{equation}
        \tilde{x} \left[ n\right]
        = 
        x \left[ n\right]
        *
        s \left[ n\right]
    \end{equation}
}

cioè una convoluzione tra $x \left[ n\right]$ e $s \left[ n\right]$, dove: 

{
    \Large 
    \begin{equation}
        s \left[ n\right] 
        = 
        \sum_{k = - \infty}^{+ \infty}
        \delta \left[ n - k N\right]
    \end{equation}
}

{
    \Large 
    \begin{equation}
        x \left[ n\right]
        = 
        \begin{cases}
            \tilde{x} \left[ n\right] \text{ per } 0 \leq n \leq N - 1 
            \\
            0 \text{ altrove}
        \end{cases}
    \end{equation}
}

\newpage 

In altre parole, una sequenza periodica $\tilde{x} \left[ n\right]$ è una convoluzione di un periodo con un treno di impulsi $s \left[ n\right]$ di periodo N. \newline 

Da un punto di vista grafico, a titolo di esempio: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.25]{DTFT di una sequenza periodica esempio.png}
\end{figure}

\subsection{DFS - Discrete Fourier series}

\footnote{ 
    Slide del prof | 7 - DFT | pag 4 - 5\\
    Appunti| 2025-10-28 | pag 4   
}

Come nel caso continuo, le sequenze periodiche possono essere rappresentate dalla serie discreta di Fourier (a.k.a. DFS - Discrete Fourier Series). \newline 

Quindi, possiamo rappresentare una qualasiasi sequenza periodica $\tilde{x} \left[ n\right]$ che si estende da $\left[ - \infty, + \infty\right]$ come: 

{
    \Large 
    \begin{equation}
        \tilde{x} \left[ n\right]
        = 
        \dfrac{1}{N}
        \sum_{k}
        \tilde{X} \left[ k\right] \cdot e^{\jmath \omega_0 k n}
    \end{equation}
}

dove: 

\begin{itemize}
    \item N è il periodo 
    \item $\dfrac{1}{N}$ è un coefficiente moltiplicativo 
    \item $\tilde{X} \left[ k\right]$ è il coefficiente della serie, cioè il k-esimo coefficiente della DFS di $\tilde{x} \left[ n\right]$
\end{itemize}

Dalla formula di $\tilde{x} \left[ n\right]$, possiamo svolgere diverse considerazioni. \newline 

Grazie anche alla notazione, dalla formula di $\tilde{x} \left[ n\right]$, possiamo notare che i coefficiente della serie DFS, 
$\tilde{X} \left[ k\right]$ sono estendibili da $\left[ - \infty, + \infty\right]$, proprio come $\tilde{x} \left[ n\right]$. \newline 

Sapendo che la pulsazione fondamentale $\omega_0$ vale: 

{
    \Large 
    \begin{equation}
        \omega_0 
        =
        \dfrac{2 \pi}{N}
    \end{equation}
}

esistono solo N distinti esponenziali complessi: 

{
    \Large 
    \begin{equation}
    \sum_{k = 0}^{N - 1}
    e^{\jmath \omega_0 k n} 
    = 
    e^{\jmath \frac{2 \pi}{N} \cdot 0}
    +
    e^{\jmath \frac{2 \pi}{N} \cdot 1}
    + 
    \dots 
    e^{\jmath \frac{2 \pi}{N} \cdot \left(N  - 1\right)}  
    \end{equation}
}

Grazie alla periodicità di $\tilde{X} \left[ k\right]$ e $\tilde{x} \left[ n\right]$, possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        e^{\jmath \frac{2 \pi}{N} \cdot 0} 
        = 
        e^{\jmath \frac{2 \pi}{N} \cdot N}
    \end{equation}
}

In generale, dato un valore di k qualasiasi, possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        e^{\jmath \frac{2 \pi}{N} \cdot k} 
        = 
        e^{\jmath \frac{2 \pi}{N} \cdot \left(k + N\right)}
    \end{equation}
}

Scrivendo $e^{- \jmath \frac{2 \pi}{N}}$ come: 

{
    \Large 
    \begin{equation}
        W_N = e^{- \jmath \frac{2 \pi}{N}}
    \end{equation}
}

allora possiamo scrivere $\tilde{x} \left[ n\right]$ come: 

{
    \Large 
    \begin{equation}
        \begin{split}
        \tilde{x} \left[ n\right]
        &= 
        \dfrac{1}{N}
        \sum_{k}
        \tilde{X} \left[ k\right] \cdot e^{\jmath \omega_0 k n}
        \\
        &\downarrow
        \\
        \tilde{x} \left[ n\right]
        &= 
        \dfrac{1}{N}
        \sum_{k}
        \tilde{X} \left[ k\right] \cdot W_N^{-k n}
        \end{split}
    \end{equation}
}

\newpage 

Il termine $W_N$ prende il nome di Twiddle. \newline 

\begin{tcolorbox}
    Il fattore di Twiddle verrà apprfondito nei futuri capitoli della FFT, ma se intanto vuoi avere un assaggio dell'importanza del Twiddle: 

    \begin{itemize}
        \item \url{https://www.linkedin.com/pulse/how-fft-algorithm-works-part-5-twiddle-factors-mark-newman/}
        \item \url{https://www.sciencedirect.com/topics/engineering/discrete-fourier-transform}
    \end{itemize}
\end{tcolorbox}

In particolare la formula: 

{
    \Large
    \begin{equation}
        \tilde{x} \left[ n\right]
        =
        \dfrac{1}{N}
        \sum_{k = 0}^{N -1}
        \tilde{X} \left[ k\right] \cdot W_N^{-k n}
    \end{equation}
}

prende il nome di DFS inversa. \newline 

Si può dimostrare anche il processo inverso, cioè la DFS diretta come: 

{
    \Large
    \begin{equation}
        \tilde{X} \left[ k\right]
        =
        \sum_{k = 0}^{N -1}
        \tilde{x} \left[ n\right] \cdot W_N^{k n}
    \end{equation}
}

ricordando che: 

{
    \Large 
    \begin{equation}
        \sum_{k = 0}^{N - 1}
        W_N^{kn} 
        = 
        \begin{cases}
            N \text{ per n multiplo di N}
            \\
            0 \text{ altrimenti}
        \end{cases}
    \end{equation}
}

Grazie a questa proprietà dei twiddle $W_N^{kn}$, possiamo dire che il twiddle forma una base ortognale dello spazio nel dominio di DFS. \newline

\subsubsection{Relazione tra DTFT e DFS}

\footnote{ 
    Slide del prof | 7 - DFT | pag 7 - 8\\
    Appunti| 2025-10-28 | pag 4   
}

Poichè una sequenza periodica $\tilde{x} \left[ n\right]$ con periodo N si può vedere come la convoluzione del singolo periodo di $x \left[ n\right]$ 
con un treno di impulsi $s \left[ n\right]$, spaziati di N, allora, possiamo scrivere che la DTFT di $x \left[ n\right]$ vale: 

{
    \Large 
    \begin{equation}
        X \left(e^{\jmath \omega}\right) 
        = 
        \sum_{n = - \infty}^{+ \infty}
        x \left[ n\right]
        \cdot 
        e^{- \jmath \omega n}
        =
        \sum_{n = 0}^{N - 1}
        x \left[ n\right]
        \cdot 
        e^{- \jmath \omega n} 
    \end{equation}
}

per cui: 

{
    \Large 
    \begin{equation}
        \tilde{X} \left[ k\right]
        = 
        \left. 
        X \left(e^{\jmath \omega}\right)
        \right|_{\omega = \dfrac{2 \pi}{N}k}
    \end{equation}
}

dove $\tilde{X} \left[ k\right]$ è la DFS di $\tilde{x} \left[ n\right]$. \newline 

Grazie a quest'ultima relazione tra $\tilde{X} \left[ k\right]$ e $X \left(e^{\jmath \omega}\right)$, 
possiamo osservare che la DFS di $\tilde{x} \left[ n\right]$, cioè $\tilde{X} \left[ k\right]$, corrisponde alla DTFT di $x \left[ n\right]$, cioè $X \left(e^{\jmath \omega}\right)$, 
campionata in N frequenze equispazionate: 

{
    \Large 
    \begin{equation}
        0, \dfrac{2 \pi}{N} \cdot 1, \dfrac{2 \pi}{N} \cdot 2, \dots, \dfrac{2 \pi}{N} \cdot \left(N - 1\right) 
    \end{equation}
}

In altri termini, eseguire la DFS corrisponde a valutare la trasformata-z della sequenza $x \left[ n\right]$, 
cioè $X \left(z\right)$, su N punti equispaziati sul cerchio unitario, come si visualizza dalla seguente figura: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.25]{Cerchio unitario discretizzato per la DFS.png}
\end{figure} 

Inoltre, si può calcolare la DTFT della sequenza $\tilde{x} \left[ n\right]$, cioè $\tilde{X} \left(e^{\jmath \omega}\right)$ come: 

{
    \Large 
    \begin{equation}
        \tilde{X} \left(e^{\jmath \omega}\right) 
        = 
        \sum_{k = - \infty}^{+ \infty}
        \dfrac{2 \pi}{N}
        \cdot 
        X \left(e^{\jmath \omega}\right)
        \cdot 
        \overline{\delta}
        \left(
            \omega 
            - 
            \dfrac{2 \pi}{N} k
        \right) 
    \end{equation}
}

dove: 

\begin{itemize}
    \item $X \left(e^{\jmath \omega}\right)$ è la trasformata di Fourier di $x \left[ n\right]$ nel singolo periodo 
    \item $\overline{\delta}
        \left(
            \omega 
            - 
            \dfrac{2 \pi}{N} k
        \right) $ è un treno di impulsi di Dirac 
\end{itemize}

\section{Sequenze aperiodiche di durata finita}

\footnote{ 
    Slide del prof | 7 - DFT | pag 9 - 10\\
    Appunti| 2025-10-28 | pag 4   
}

Data una sequenza periodica $\tilde{x} \left[ n\right]$, essa è completamente rappresentata dalla sua sequenza DFS $\tilde{X} \left[ k\right]$ 
con $k = 0, 1, \dots, N-1$. \newline 

Invecem data una sequenza aperiodica $x \left[ n\right]$ di lunghezza N, 
quest'ultima può essere resa periodica replicandola sull'asse n, proprio come si vede nelle figure di esempio: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.5]{Confronto tra una sequenza aperiodica e periodica di esempio.png}
\end{figure} 

Allora, la DFS di $\tilde{x} \left[ n\right]$ corrisponde alla DTFT di $x \left[ n\right]$ campionata in N frequenze equispaziate, 
quindi, per conoscere la DFS non bisogna utilizzare la DTFT. \newline 

\begin{tcolorbox}
    Ma tanto, ingegneristicamente parlando, non la utilizzeremo mai la DFS. 
\end{tcolorbox}

Quindi, possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        x \left[ n\right]  
        \leftrightarrow 
        \left.
        X \left(e^{\jmath \omega}\right)
        \right|_{\omega = \frac{2 \pi}{N} k}
    \end{equation}
}

per $n = 0, 1, \dots, N-1$ e $k = 0, 1, \dots, N-1$

\section{DFT - Discrete Fourier Transform}

\footnote{ 
    Slide del prof | 7 - DFT | pag 15\\
    Appunti| 2025-10-29 | pag 1
}

Una sequenza aperiodica $x \left[ n\right]$ di lunghezza N può essere rappresentata completamente (o anche detta ricostruita) dai suoi N campioni per 
i seguenti valori di $\omega$: 

{
    \Large 
    \begin{equation}
        \omega = \dfrac{2 \pi}{N} \cdot 0, \dfrac{2 \pi}{N} \cdot 1, \dfrac{2 \pi}{N} \cdot 2, \dots, \dfrac{2 \pi}{N} \cdot \left(N -1\right) 
    \end{equation}
}

dalla sua trasformata DTFT, cioè $X \left(e^{\jmath \omega}\right)$. \newline 

Si definisce periò la DFT diretta come: 

{
    \Large 
    \begin{equation}
        X \left[ k\right]
        = 
        \begin{cases}
            \sum_{n = 0}^{N -1}
            x \left[ n\right] \cdot W_N^{kn} \quad \text{ per } 0 \leq k \leq N -1 
            \\
            0 \quad \text{ altrove}
        \end{cases}
    \end{equation}
}

\begin{tcolorbox}
    Ricordiamo che il Twiddle $W_N$ vale: 
{
    \Large 
    \begin{equation}
        W_N = e^{- \jmath \frac{2 \pi}{N}}
    \end{equation}
}
\end{tcolorbox}

Possiamo definire il processo inverso della DFT diretta, cioè la DFT inversa come: 

{
    \Large 
    \begin{equation}
        x \left[ n\right]
        = 
        \begin{cases}
            \sum_{k = 0}^{N -1}
            X \left[ k\right] \cdot W_N^{-kn} \quad \text{ per } 0 \leq k \leq N -1 
            \\
            0 \quad \text{ altrove}
        \end{cases}
    \end{equation}
}

Possiamo svolgere le seguenti osservazioni: 
\begin{itemize}

    \item La DFT ha le stesse proprietà della DFS (che non abbiamo elencato in questi appunti), ma perde la periodicità della DFS. \newline 

\begin{tcolorbox}
    Perdere la periodicità della DFS è cosa buona e giusta perchè sennò, per calcolare la sequenza, si impiegerebbe un tempo che va da $- \infty$ a $+ \infty$, 
    cioè un tempo infinito. 
\end{tcolorbox}

    \item La DFT di una sequenza finita di lunghezza N è una sequenza di lunghezza finita N 
    \item La DFT equivale ad un periodo della DFS della sequenza periodica $\tilde{x} \left[ n\right]$, quest'ultima ottenuta replicando $x \left[ n\right]$ per periodi multipli di N da $- \infty$ a $+ \infty$
\end{itemize}


\begin{tcolorbox}
    QUindi possiamo dire che la DFT è la trasformata di Fourier della sequenza $x \left[ n\right]$ in N punti. 
\end{tcolorbox}

\subsection{Relazione tra DFT e DFS}

\footnote{ 
    Slide del prof | 7 - DFT | pag 16\\
    Appunti| 2025-10-29 | pag 1
}

Per passare da una sequenza periodica $\tilde{x} \left[ n\right]$ a una seuqnza aperiodica $x \left[ n\right]$, 
dobbiamo utilizzare la seguente formula: 

{
    \Large 
    \begin{equation}
        \tilde{x} \left[n\right]
        = 
        x \left[ \left(n \right)_{\mod N}\right]
    \end{equation}
}

dove la dicitura $\mod N$ si intende il modulo, cioè il resto intero della divisione tra l'indice n e N periodo. \newline 

Se invece volessimo fare il processo inverso: 

{
    \Large 
    \begin{equation}
        x \left[ n\right]
        = 
        \begin{cases}
            \tilde{x} \left[ n\right] \quad \text{ per } 0 \leq n \leq N -1
            \\
            0 \quad \text{ altrove }
        \end{cases}
    \end{equation}
}

Per quanti riguarda le rispettive rappresentazioni nello spettro, cioè per passare da una DFT a una DFS, bisogna applicare questa formula: 

{
    \Large 
    \begin{equation}
        \tilde{X} \left[k\right]
        = 
        X \left[ \left(k \right)_{\mod N}\right]
    \end{equation}
}

Se invece volessimo fare il processo inverso: 

{
    \Large 
    \begin{equation}
        X \left[ k\right]
        = 
        \begin{cases}
            \tilde{X} \left[ k\right] \quad \text{ per } 0 \leq k \leq N -1
            \\
            0 \quad \text{ altrove }
        \end{cases}
    \end{equation}
}

\subsection{Zero Padding}

\footnote{ 
    Slide del prof | 7 - DFT | pag 17\\
    Appunti| 2025-10-29 | pag 1
}

Poichè la DFT della sequenza $x \left[ n\right]$ è la DTFT di $x \left[ n\right]$ campionata nei punti: 

{
    \Large 
    \begin{equation}
        \omega = \dfrac{2 \pi}{N} \cdot 0, \dfrac{2 \pi}{N} \cdot 1, \dfrac{2 \pi}{N} \cdot 2, \dots, \dfrac{2 \pi}{N} \cdot \left(N -1\right) 
    \end{equation}
}

e poichè la ricostruzione perfetto di $x \left[ n\right]$ richiede \textbf{ almeno } N campioni in frequenza, per evitare aliasing temporale 
(come visto in \ref{sec: Aliasing nel tempo} \nameref{sec: Aliasing nel tempo} a pag \pageref{sec: Aliasing nel tempo}), 
allora si può aumentare la lunghezza della DFT con la tecnica chiamata Zero Padding, cioè aggiungere M-N zeri alla sequenza iniziale $x \left[ n\right]$. \newline 

\begin{tcolorbox}
    Alcune accortezze riguardo allo Zero Padding. \newline 

    La prima accortezza è che questa tecnica può essere applicata o prima o dopo che si ha la sequenza $x \left[ n\right]$, 
    cioè non si può aggiungere degli M - N zeri all'interno della sequenza $x \left[ n\right]$ (a meno che $x \left[ n\right]$ non abbia dei valori uguali a zero prima della tecnica dello zero padding, in quel caso è giusto che siano presenti degli zeri). \newline 

    La seconda accortezza è che la tecnica Zero Padding va scelta con cura in base al segnale perchè si aggiunge "artificialmente" dei valori che non esistono nella sequenza inziale $x \left[ n\right]$.  
\end{tcolorbox}

Come si vede dalla seguenti figure di esempio: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.6]{Zero padding esempio.png}
\end{figure} 

dove si confronta: 

\begin{itemize}
    \item in alto, una sequenza $x \left[ n\right]$ senza zero padding, perchè $M = N = 5$ 
    \item in basso, una sequenza $x \left[ n\right]$ con zero padding, perchè si è aggiunto $M - N = 10 - 5 = 5$ zeri, quindi $M > N$
\end{itemize}
 
con a destra i relativi spettri dei moduli della DFT. \newline 

La reversibilità tra il dominio n e quello delle DFT è garantito perchè, in tutti e due i casi, $M \ge N$. \newline 

\newpage 

Confrontando gli spettri delle DFT delle due sequenze (cioè le due figure a destra) possiamo notare che: 

\begin{itemize}
    \item la sequenza rossa tratteggiata indica lo spettro della DFT se M tende ad infinito, cioè se la DFT fosse uguale alla DTFT 
    \item nella sequenza con zero padding, cioè quella con $M = 10$, ci sono più valori di $\abs{X \left[ k\right]}$ rispetto alla sequenza senza zero padding, con $M = 5$; questi valori in più non altera la DFT, ma aggiunge campioni intermedi (chiamati anche interpolazione)
\end{itemize}

\begin{tcolorbox}
    La scelta dei valori M deve essere almeno uguale a N. \newline 

    Se si vuole elaborare il segnale $X \left[ k\right]$, è sempre meglio scegliere un M più piccolo possibile. \newline 

    Se invece si vuole stimare meglio la DTFT, quindi avvicinarsi alla linea tratteggiata rossa in figura, bisogna scegliere un valore di M più alto possibile. 
\end{tcolorbox}

\subsection{Proprietà della DFT}

\footnote{ 
    Slide del prof | 7 - DFT | pag 18\\
    Appunti| 2025-10-29 | pag 1 - 2
}

La DFT della sequenza di $x \left[ n\right]$ non è altro che la DFS della sequenza periodicizzata $\tilde{x} \left[ n\right]$, 
quindi le proprietà della DFT sono le stessa della DFS, salvo che ora gli indici vanno calcolati in modulo N. \newline 

Le proprietà della DFT sono le seguenti: 

\begin{itemize}
    \item linearità 
    \item dualità 
    \item simmetria 
    \item traslazione, sia in tempo che in frequenza
\end{itemize}

Per quanto riguarda la traslazione nel tempo, avremo che: 

{
    \Large 
    \begin{equation}
        x \left[ \left(n - m\right)_{N}\right]
        \leftrightarrow 
        X \left[ k\right] \cdot W_N^{km}
    \end{equation}
}

dove il simbolo N come pedice a $\left( n - m\right)$ indica una traslazione (in inglese) shift, circolare, 
cioè ciò che esce da un altro rientra nell'altro. \newline 

Un esempio di traslazione circolare nel tempo con $m = 3$ ed $N = 6$ (con note):

\subsection{Convoluzione periodica}

\footnote{ 
    Slide del prof | 7 - DFT | pag 21\\
    Appunti| 2025-10-29 | pag 2 - 3
}

\begin{tcolorbox}
    Se abbiamo una convoluzione lineare tra due sequenze $x_1 \left[ n\right]$ lunga L e $x_2 \left[ n\right]$ lunga M 
    come studiato in \ref{subsec: Prodotto di convoluzione e le sue proprietà} \nameref{subsec: Prodotto di convoluzione e le sue proprietà} a pag \pageref{subsec: Prodotto di convoluzione e le sue proprietà}, 
    avremo che la sequenza $x_1 \left[ n\right] * x_2 \left[ n\right]$ sarò lunga $L + M -1$, quindi la sequenza convoluita sarà più lunga di $x_1 \left[ n\right]$ e $x_2 \left[ n\right]$. \newline 
    
    Ma, nel "fantastico mondo del DSP", noi vorremmo avere una convoluzione di lunghezza fissa: per questo motivo, oltre alla convoluzione lineare precedentemente studiata, 
    è necessario presentare altri tipi di convoluzione. 
\end{tcolorbox}

Date due sequenze periodiche $\tilde{x_1} \left[ n\right]$ e $\tilde{x_2} \left[ n\right]$ con entrambe periodo N, 
allora si può definire la loro convoluzione circolare come: 

{
    \Large 
    \begin{equation}
        \tilde{x}_1 \left[ n\right] 
        \circledast
        \tilde{x}_2 \left[ n\right] 
    \end{equation}
}

dove il simbolo $\circledast$ indica il simbolo per la convoluzione circolare. \newline 

La convoluzione circolare tra due sequenze è uguale a: 

{
    \Large 
    \begin{equation}
        \begin{split}
        \tilde{x}_1 \left[ n\right] 
        \circledast
        \tilde{x}_2 \left[ n\right]
        &= 
        \sum_{m = 0}^{N -1}
        \tilde{x}_1 \left[ m\right] 
        \cdot
        \tilde{x}_2 \left[ n - m\right]
        \\
        &= 
        \sum_{m = 0}^{N -1}
        \tilde{x}_1 \left[ n - m\right] 
        \cdot
        \tilde{x}_2 \left[ m\right]  
        \end{split}
    \end{equation}
}

dove $\tilde{x}_1 \left[ n - m\right] $ e $\tilde{x}_2 \left[ n - m\right]$ si intendono degli shift lineari 
(quindi la sequenza $\tilde{x}_1 \left[ n\right]$ o $\tilde{x}_2 \left[ n\right]$ viene ritardata N volte e poi, ad ogni ritardo, la sequenza shiftata viene moltiplicata per l'altra sequenza e poi vengono sommati i componenti). \newline 

Le due sommatorie sono uguali perchè la convoluzione ciroclare è commutativa. \newline 

Un esempio di convoluzuione circolare con sequenze periodiche è la seguente: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.4]{Esempio convoluzione circolare con sequenze periodiche.png}
\end{figure} 

\newpage

Si può dimostrare (cioè non lo dimostreremo) che valgono le seguenti relazioni tra dominio n e dominio DFS se tutte le sequenze sono di periodo N: 

{
    \Large 
    \begin{equation}
        \tilde{x}_1 \left[ n\right]
        \circledast
        \tilde{x}_2 \left[ n\right]
        \leftrightarrow
        \tilde{X}_1 \left[ k\right]
        \cdot
        \tilde{X}_2 \left[ k\right]
    \end{equation}
}

cioè la convoluzione circolare tra $\tilde{x}_1 \left[ n\right]$ e $\tilde{x}_2 \left[ n\right]$ è un prodotto nel dominio DFS, invece: 

{
    \Large 
    \begin{equation}
        \tilde{x}_1 \left[ n\right]
        \cdot
        \tilde{x}_2 \left[ n\right]
        \leftrightarrow
        \dfrac{1}{N}
        \left(
        \tilde{X}_1 \left[ k\right]
        \circledast
        \tilde{X}_2 \left[ k\right]
        \right)
    \end{equation}
}

cioè il prodotto tra $\tilde{x}_1 \left[ n\right]$ e $\tilde{x}_2 \left[ n\right]$ è una convoluzione circolare nel dominio DFS. \newline 

Siccome abbiamo a che fare con sequenze periodiche, la convoluzione circolare, in questo caso, prende il nome di convoluzione periodica. \newline

\subsection{Convoluzione circolare}

\footnote{ 
    Slide del prof | 7 - DFT | pag 22\\
    Appunti| 2025-10-29 | pag 3 - 4
}

La convoluzione circolare è una convoluzione periodica svolta per sequenze non periodiche. \newline 

Date due sequenze $x_1 \left[ n\right]$ e $x_2 \left[ n\right]$ di lunghezza N, allora possiamo definire la loro convoluzione circolare come: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n\right]
        \circledast
        x_2 \left[ n\right]
        = 
        \sum_{m = 0}^{N -1}
        x_1 \left[ m\right]
        \cdot
        x_2 \left[ \left( n- m\right)_{N}\right]  
    \end{equation}
}

dove la N sotto a $\left(n - m\right)$ indica il modulo N dell'indice. \newline 

Nel dominio DFT, la convoluzione circolare $x_1 \left[ n\right]$ e $x_2 \left[ n\right]$ nel dominio n vale: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n\right]
        \circledast
        x_2 \left[ n\right]
        \leftrightarrow
        X_1 \left[ k\right]
        \cdot
        X_2 \left[ k\right]  
    \end{equation}
}

Un esempio di convoluzuione circolare con sequenze aperiodiche è la seguente: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.45]{Esempio convoluzione circolare con sequenze aperiodiche.png}
\end{figure} 

Nel dominio DFT, il prodotto tra due sequenze $x_1 \left[ n\right]$ e $x_2 \left[ n\right]$ nel dominio n vale: 
        
{
    \Large 
    \begin{equation}
        x_1 \left[ n\right]
        \cdot
        x_2 \left[ n\right]
        \leftrightarrow
        \dfrac{1}{N}
        \left(
        X_1 \left[ k\right]
        \circledast
        X_2 \left[ k\right] 
        \right)
        \end{equation}
}

dove $ \dfrac{1}{N} \left( X_1 \left[ k\right] \circledast X_2 \left[ k\right] \right)$ vale: 

{
    \Large 
    \begin{equation}
        \dfrac{1}{N}
        \left(
        X_1 \left[ k\right]
        \circledast
        X_2 \left[ k\right] 
        \right)
        =
        \dfrac{1}{N}
        \sum_{m = 0}^{N - 1}
        X_1 \left[ m\right]
        \cdot
        X_2 \left[ \left( k- m\right)_{N}\right]  
    \end{equation}
}

\subsection{Rapporto tra convoluzuioni aperiodiche e circolare}

\footnote{ 
    Slide del prof | 7 - DFT | pag 23 - 26\\
    Appunti| 2025-11-04 | pag 1 - 2
}

Esistono diverse relazioni tra una convoluzione aperiodica e quella circolare: 
di seguito elenchiamo due casi di studio particolare. \newline 

\begin{itemize}

\item \textbf{$x_1 \left[ n \right]$ e $x_2 \left[ n \right]$ hanno lunghezza finita N} \newline 

La prima è che, se le due sequenze $x_1 \left[ n \right]$ e $x_2 \left[ n \right]$ hanno lunghrzza finita N, 
allora possiamo scrivere che la loro convoluzione lineare è uguale a: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        *
        x_2 \left[ n \right] 
        = 
        \sum_{m = 0}^{N - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2 \left[ n - m \right]
    \end{equation}
}

convoluzine nel dominio n che, facendo la DTFT (e non la DFT), è uguale a: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        *
        x_2 \left[ n \right] 
        \leftrightarrow
        X_1 \left(e^{\jmath \omega}\right)
        \cdot
        X_2 \left(e^{\jmath \omega}\right) 
    \end{equation}
}

Invece facendo la convoluzione ciclica tra le due sequenze $x_1 \left[ n \right]$ e $x_2 \left[ n \right]$, 
abbiamo che: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        \circledast
        x_2 \left[ n \right]
        = 
        \sum_{m = 0}^{N - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2 \left[ \left(n - m \right)_{N} \right]
    \end{equation}
}

convoluzione circolare nel dominio n che, facendo la DFT (e non la DTFT), è uguale a: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        \circledast
        x_2 \left[ n \right] 
        \leftrightarrow
        X_1 \left[ k\right]
        \cdot
        X_2 \left[ k\right] 
    \end{equation}
}

La convoluzione aperiodica $ x_1 \left[ n \right] * x_2 \left[ n \right] $ è lunga $2 N - 1$, 
grazie agli m shift che deve fare: 

{
    \Large 
    \begin{equation}
        \sum_{m = 0}^{N - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2 \left[ n - m \right]
    \end{equation}
}

invece, nella convoluzione circolare $ x_1 \left[ n \right] \circledast x_2 \left[ n \right] $ è lunga N, 
che è la stessa lunghezza sia di $ x_1 \left[ n \right]$ che di $ x_2 \left[ n \right]$ perchè gli m shift sono svolti in modulo N: 

{
    \Large 
    \begin{equation}
        \sum_{m = 0}^{N - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2 \left[ \left(n - m \right)_{N} \right]
    \end{equation}
}

Questo è un enorme vantaggio di una convoluzione ciclica. \newline 

Inoltre, nella convoluzione aperiodica nel dominio n, è necessario svolgere una DTFT, che, è periodica nello spettro, 
invece in una convoluzione ciclica nel dominio n, è necessario svolgere una DFT, che non è periodica e le sequenze dipendono dagli indici k-esimi. \newline 

In generale, la convoluzione aperiodica e quella ciclica non coincidono. \newline 

\newpage 

Grazie a questa figura di esempio: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.45]{convoluzione ciclica come convoluzione aperiodica con aliasing esempio.png}
\end{figure} 

possiamo notare che la convoluzione ciclica è come una convoluzione aperiodica con aliasing. \newline 

Se consideriamo la lunghezza di una convoluzione ciclica lunga N (che nella figura sono i pallini che vanno da 0 a +N), 
allora i contributi di lunghezza maggiore della convoluzione ciclica si sommano (che nella figura i pallini rossi si sommano ai pallini blu), 
creando appunto aliasing. \newline 

La somma tra gli impulsi non è reversibilie. \newline 

Per evitare aliasing, e che quindi non si sommano i contributi nella stessa finistra finita, 
possiamo svolgere uno zero padding alle due sequenze, affinche si arriva a sequenze di almeno $2N - 1$ punti. \newline 

Quindi una convoluzione aperiodica è uguale alla convoluzione ciclica se e solo se si svolge zero padding a $2 N - 1$ punti e si svolge una DFT a $2 N - 1$ punti. \newline

Questo è un altro vantaggio dello zero padding e della DFT 
perchè permette di calacolare la DTFT attraverso la DFT, in particolare grazie al prodotto delle due sequenze. \newline 

\newpage 

\item \textbf{$x_1 \left[ n \right]$ e $x_2 \left[ n \right]$ hanno lunghezza finita diversa} \newline 

Se abbiamo che due sequenze: 

\begin{itemize}
    \item $x_1 \left[ n\right]$ aperiodica di lunghezza L 
    \item $x_2 \left[ n\right]$ aperiodica di lunghezza P, in cui $P < L$, cioè $P \neq L$
\end{itemize}

allora abbiamo che la convoluzione aperiodica nel dominio n: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        *
        x_2 \left[ n \right] 
        = 
        \sum_{m = 0}^{L - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2 \left[ n - m \right]
    \end{equation}
}

sarà lunga $ L+ P - 1$. \newline 

Nella sommatoria, l'indice m parte da 0 e finisce a $L - 1$ perchè si fa la convoluzione per la lunghezza della sequenza più lunga (che in questo caso è $x_1 \left[ n\right]$). \newline 

Se invece facessimo una convoluzione ciclica nel dominio n: 

{
    \Large 
    \begin{equation}
        x_1 \left[ n \right]
        \circledast
        x_2^{'} \left[ n \right]
        = 
        \sum_{m = 0}^{L - 1}
        x_1 \left[ m \right]
        \cdot 
        x_2^{'} \left[ \left(n - m \right)_{N} \right]
    \end{equation}
}

dove $x_2^{'} \left[ n \right]$ è la sequenza $x_2 \left[ n \right]$ con zero-padding fino ad $L - 1$, 
allora la sequenza circolare è lunga L, che è di lunghezza minore, cioè $ L+ P - 1$, della convoluzine aperiodica . \newline 

Come il caso precedente in cui $x_1 \left[ n \right]$ e $x_2 \left[ n \right]$, hanno la stessa lunghezza N, 
a titolo di esempio abbiamo la seguente figura: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.4]{convoluzione ciclica come convoluzione aperiodica con aliasing esempio lunghezze diverse.png}
\end{figure} 

e ci sono gli stessi problemi di aliasing a causa delle lunghezze delle sequenze. \newline 

Ma se le DFT sono fatte ad almeno $L + P - 1 $ punti, mediante zero padding, 
allora la convoluzione ciclica e quella aperiodica coincidono. \newline 

\end{itemize}
