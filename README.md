## 🔬 Un assaggio di Teoria

Dalle comunicazioni digitali all'elaborazione audio, il cuore pulsante del DSP è la **Trasformata Discreta di Fourier (DFT)**:

$$X[k] = \sum_{n=0}^{N-1} x[n] \cdot e^{-j \frac{2\pi}{N} kn}$$
