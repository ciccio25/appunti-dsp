\subsubsection{Sequenza impulso unitario}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 2 \\
    Appunti | 2025-09-24 | pag 3
}

La sequenza elementare alla base di tutte le sequenze è la sequenza impulso unitario. \newline 

Di seguito dal punto di vista grafico la sequenza impulso unitario: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Sequenza impulso unitario.png}
\end{figure}  

La sequenza $\delta [n]$ impulso unitario gode della sequenza proprietà: 

{
    \Large 
    \begin{equation}
        \delta \left[ n \right] 
        = 
        \begin{cases}
            0 \text{ per } n \neq 0 
            \\
            1 \text{ per } n = 0 
        \end{cases}
    \end{equation}
}

cioè, scritta a parole, vale 1 per n e vale 0 per tutti gli altri valori di n. \newline 

\begin{tcolorbox}
    Se avete svolto e/o seguito l'esame di Teoria dei Segnali (o Segnali Determinati e Aleatori) con Franco Chiaraluce, 
    la sequenza impulso unitario $\delta [n]$ NON è la Delta di Dirac $\delta (t)$, dove: 
    
    {
        \Large 
        \begin{equation}
            \delta (t)
            = 
            \begin{cases}
                0 \text{ per } t \neq 0
                \\
                + \infty \text{ per } t_0 = 0
            \end{cases}
        \end{equation}
    }

    di cui gode della sequente proprietà: 

    {
        \Large 
        \begin{equation}
            \int_{- \infty}^{+\infty} \delta (t) dt = 1
        \end{equation}
    }

    n è l'indice della sequenza. 

\end{tcolorbox}

Questo tipo di sequenza non pone problemi di definizione e realizzabilità. \newline 

\newpage 

\subsubsection{Sequenza impulso unitario ritardato}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 2 \\
    Appunti | 2025-09-24 | pag 3 \\ 
    Latex | 2025-09-30 Mattino | pag 9
}

Un altro tipo di sequenza, che proviene dalla sequenza impulso unitario, 
è la sequenza impulso unitario ritardato $\delta \left[n - n_0\right]$. \newline 

Dal punto di vista matematico, la sequenza impulso unitario ritardato è la sequente: 

{
    \Large 
    \begin{equation}
        \delta \left[n - n_0\right] 
        = 
        \begin{cases}
            0 \text{ per } n \neq n_0 \\
            1 \text{ per } n = n_0
        \end{cases}
    \end{equation}
}

cioè, è la sequenza impulso unitario $\delta \left[ n \right]$ ritardato all'indice $n_0$. \newline 

Se moltiplichiamo la sequenza impulso unitario $\delta \left[ n \right]$ per un coefficiente A 
e lo ritardiamo all'indice $n_0$: 

{
    \Large 
    \begin{equation}
        n_0 = 3
    \end{equation}
}

allora ottieniamo la sequente sequenza impulso ritardato: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Sequenza impulso ritardato.png}
\end{figure}   

Dal punto di vista matematico: 

{
    \Large 
    \begin{equation}
        \delta \left[n - 3\right] 
        = 
        \begin{cases}
            0 \text{ per } n \neq n_0 \\
            A \text{ per } n = 3
        \end{cases}
    \end{equation}
}

Quest'ultima formula ci dice che, se la sequenza impulso unitario viene moltiplicata per un coefficiente moltiplicativo A, 
la sequenza impulso unitario rimane tale se $A = 1$, altrimenti è una sequenza impulso. \newline 

Il ritardo può essere definito anche in inglese come shift o delay. \newline 

Lo shift è una proprietà di traslazione di sequenze. \newline

\subsection{Proprietà di campionamento}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 3 \\
    Appunti | 2025-09-24 | pag 2
}

Di seguito la proprietà di campionamento. \newline 

Una qualsiasi sequenza $x \left[ n \right]$ può essere espressa come somma di impulsi unitari 
oppurtunamente ritardati e amplificati. \newline 

Da un punto di vista matematico, possiamo scrivere la sequenza $x \left[ n \right]$ come: 

{
    \Large 
    \begin{equation}
        x \left[ n \right] 
        = 
        \sum_{k = - \infty}^{+ \infty }
        x \left[ k \right] \cdot \delta \left[ n - k \right]
    \end{equation}
}

Applicando la proprietà di campionamento, 
possiamo visualizzare la sequenza $x \left[ n \right]$: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Sequenza da campionare.png}
\end{figure}   

nelle sue componenti cioè come somma dei seguenti componenti: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Segnale campionato.png}
\end{figure}   

\begin{tcolorbox}
    Rispetto al corso di Teoria dei Segnali, in questo caso si indica la proprietà di campionamento, 
    che non è il teorema del campionamento: 
    sono due cose differenti. \newline 
    
    Il teorema del campionamento afferma una rappresentazione, quindi una funzione biunivoca (cioè un segnale può essere visto sia nel dominio del tempo che in quello delle frequenze). \newline 

    Invece, con la proprietà del campionamento, non si cambia dominio: si rimane sempre nello stesso dominio della sequenza originale (che in questo caso è quello degli indici discreti n). 
\end{tcolorbox}

\subsubsection{Gradino unitario}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 4 \\
    Appunti | 2025-09-24 | pag 2
}

La sequenza $u \left[ n \right]$ gradino unitario ha la sequenza espressione: 

{
    \Large 
    \begin{equation}
        u \left[ n \right] 
        = 
        \begin{cases}
            0 \text{ per } n < 0
            \\
            1 \text{ per } n \geq 0
        \end{cases} 
    \end{equation}
}

Da un punto di vista grafico, 
possiamo visualizzare il gradino unitario $u \left[ n \right] $ come: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Gradino unitario.png}
\end{figure}   

Utilizzando la proprietà dell'impulso, 
possiamo scrivere il gradino unitario $u \left[ n \right] $ come: 

{
    \Large 
    \begin{equation}
        \begin{split}
            u \left[ n \right] 
            &= 
            \sum_{k = - \infty }^{+ \infty}
            u \left[ k \right]
            \cdot 
            \delta \left[ n - k \right]
            \\
            &=
            \sum_{k = - \infty }^{+ \infty}
            \delta \left[ n - k \right] 
            \\
            &=
            \sum_{k = 0}^{+ \infty}
            \delta \left[ n - k \right] 
        \end{split}
    \end{equation}
}

Si può applicare le seguenti semplificazione perchè: 

{
    \Large 
    \begin{equation}
            u \left[ k \right] = 1 \text{ } \forall k \geq 0
    \end{equation}
}

\begin{tcolorbox}
    Nel campo dei discreti, 
    non esiste l'operatore matematico dell'integrale (e che quindi calcola l'area sotto la funzione); 
    bensì si utilizzera la somma.  
\end{tcolorbox}

\newpage 

Si può verificare che: 

{
    \Large 
    \begin{equation}
        \begin{cases}
        u \left[ n \right] = \sum_{k = 0}^{n} \delta \left[ k \right] 
        \\
        \delta \left[ n \right] = u \left[ n \right] - u \left[ n - 1 \right] 
        \end{cases}
    \end{equation}
}

dove $\delta \left[ n \right]$ è l'impulso unitario. \newline 

\begin{tcolorbox}
    Nel campo dei discreti, 
    non esiste l'operatore matematico della derivata (e che quindi calcola la differenza infinitesimale tra due punti di una funzione); 
    bensì si utilizzera la differenza tra una sequenza e la sequenza stessa shiftata / ritardata di un punto.  
\end{tcolorbox}

Da un punto di vista grafico, 
se ci poniamo nell'indice $n = 0$, 
l'equazione precedente possiamo visualizzarla graficamente: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Impulso unitario come differenza di due gradini unitari.png}
\end{figure}   

Più in generale, 
è possibile definire una famiglia di sequenze mediante operazioni di sommatoria e differenza finita, 
analogamente al caso continuo. \newline

\subsubsection{Rampa unitaria}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 5 \\
    Appunti | 2025-09-24 | pag 2 \\ 
    Latex | 2025-09-30 Mattino | pag 13
}

Possiamo definire la rampa unitaria $r \left[ n \right]$ come: 

{
    \Large 
    \begin{equation}
        \begin{cases}
            r \left[ n \right] 
            = 
            \sum_{k = - \infty}^{n} 
            u \left[ k \right]
            \\ 
            u \left[ n \right] 
            = 
            r \left[ n \right]
            - 
            r \left[ n - 1\right]
        \end{cases}
    \end{equation}
}

Confrontando la rampa unitaria $r \left[ n \right]$ con il mondo analogico, 
avremo la seguente figura: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.5]{Rampa unitaria con la differenza della rampa continua.png}
\end{figure}   

La rampa unitaria $r \left[ n \right]$ viene definita unitaria perchè, per qualsiasi valore n scelto, 
la differenza (cioè la "derivata" tra un indice ed un altro) è sempre unitaria. \newline 

In formule: 

{
    \Large 
    \begin{equation}
        u \left[ n \right] = 1 \text{ } \forall n
    \end{equation}
}

\subsubsection{Sequenze esponenziali complesse}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 7 \\
    Appunti | 2025-09-24 | pag 4 \\
    Latex | 2025-09-30 Mattino | pag 15
}

Di seguito la definizione di sequenza esponenziale complessa. \newline 

Una sequenza $x \left[ n \right]$ esponenziale complessa ha la seguente formulazione matematica: 

{
    \Large 
    \begin{equation}
        x \left[ n \right] 
        = 
        A \cdot \alpha^{n} 
        \text{ con }
        A, \alpha \in \mathbb{C} 
    \end{equation}
}

L'inviluppo della sequenza ha lo stesso andamento dell'esponenziale reale. \newline 

\begin{tcolorbox}
Per inviluppo si intende "l'integrale" della sequenza. \newline 

Avendo a che fare con una sequenza sinusoidale (perchè si considerano $A$ e $\alpha$ complessi), 
allora avremo una sequenza sinusoidale, il cui andamento tende alle figure del sottocapitolo delle "Sequenze esponenziali reali". \newline 

Inoltre, essendo una sequenza complessa, e quindi sinusoidale, dovrà essere studiata sia per il modulo che per la fase. 
\end{tcolorbox}

In base ai valori di $\alpha$, 
avremo l'andamento l'andamento di $x \left[ n \right]$: 

\begin{itemize}
    \item $\abs{\alpha} < 1 $, allora $x \left[ n \right]$ è decrescente 
    \item $\abs{\alpha} = 1 $, allora $x \left[ n \right]$ è costante 
    \item $\abs{\alpha} > 1 $, allora $x \left[ n \right]$ è crescente
\end{itemize}

Possiamo definire i seguenti casi particolari delle sequenza esponanziali complesse. \newline 

Definiamo l'esponenziale complesso puro $x \left[ n \right]$ come: 

{
    \Large 
    \begin{equation}
        x \left[ n \right]
        =
        \abs{A} \cdot e^{\jmath \left( \omega n + \phi\right)}
    \end{equation}
}

Definiamo il coseno reale $x \left[ n \right]$, 
grazie alla formula di Eulero, come: 

{
    \Large 
    \begin{equation}
        \begin{split}
        x \left[ n \right]
        &=
        \cos(\omega n + \phi)
        \\
        &=
        \dfrac{e^{\jmath \left( \omega n + \phi\right)} + e^{-\jmath \left( \omega n + \phi\right)}}{2}
        \end{split}
    \end{equation}
}

Definiamo il seno reale $x \left[ n \right]$, 
grazie alla formula di Eulero, come: 

{
    \Large 
    \begin{equation}
        \begin{split}
        x \left[ n \right]
        &=
        \sin(\omega n + \phi)
        \\
        &=
        \dfrac{e^{\jmath \left( \omega n + \phi\right)} - e^{-\jmath \left( \omega n + \phi\right)}}{2 \jmath}
        \end{split}
    \end{equation}

}

\subsection{Sequenze periodiche}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 8 \\
    Appunti | 2025-09-24 | pag 4 \\ 
    Latex | 2025-09-30 Mattino | pag 16
}

Per sequenze periodiche si intendono delle sequenze $x \left[ n \right]$ che hanno la sequente proprietà: 

{
    \Large 
    \begin{equation}
        x \left[ n \right] 
        = 
        x \left[ n + N\right] 
    \end{equation}
}

dove N è un numero intero che indica il periodo. \newline 

Un tipo di sequenza periodica è l'esponenziale complesso puro: 

{
    \Large 
    \begin{equation}
        e^{\jmath \left( \omega n + \phi\right)}
    \end{equation}
}

in cui: 

\begin{itemize}
    \item $\omega$ è la pulsazione, la cui relazione con il mondo fisico si misura in $\dfrac{rad}{s}$ 
    \item $f = \dfrac{\omega}{2 \pi}$ è la frequenza, la cui relazione con il mondo fisico si misura in Hz 
    \item $\phi$ è la fase iniziale nel periodo
\end{itemize}

\begin{tcolorbox}
    Essendo nel mondo del DSP, quindi delle sequenze e non dei segnali, le sequenze stesse non hanno una grandezza fisica. \newline 

    Qui sono solo riportate le relazioni dei coefficienti astratti con il mondo reale.
\end{tcolorbox}

\newpage 

\subsubsection{Proprietà dell'esponenziale complesso puro}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 8 - 9\\
    Appunti | 2025-09-24 | pag 4 \\ 
    Latex | 2025-09-30 Mattino | pag 17
}

L'esponenziale complesso puro $e^{\jmath \left( \omega n + \phi\right)}$ gode delle seguenti proprietà. \newline 

\textbf{$1^{\circ}$ proprietà: } \newline 

Una sequenza con pulsazione $\omega_0$ ed una con $\omega_0 + 2 \pi$ sono identiche perchè: 

{
    \Large 
    \begin{equation}
        A \cdot e^{\jmath \left( \omega_0 n + 2 \pi n \phi\right)}
        = 
        A \cdot e^{\jmath \left( \omega_0 n + \phi\right)} \cdot e^{\jmath 2 \pi n} 
    \end{equation}
}

dove lo spettro è periodico con periodo $2 \pi$. \newline 

Si considerano equivalentemente: 

{
    \Large 
    \begin{equation}
        0 \leq \omega \leq 2 \pi 
        \text{ oppure }
        - \pi \leq \omega \leq + \pi
    \end{equation}
}

{
    \Large 
    \begin{equation}
        0 \leq f \leq 1 
        \text{ oppure }
        -0.5 \leq f \leq +0.5
    \end{equation}
}

perchè, grazie alla simmetricità dovuta alla periodicità, 
si può studiare mezzo periodo della sequenza. \newline 

\textbf{$2^{\circ}$ proprietà: } \newline 

Le "alte" frequenze sono intorno a $\pm \pi$, le "basse" frequenze into allo 

\begin{tcolorbox}[colback=green!5!white,colframe=green!75!black]
TO-DO: Fare il disegno delle alte e basse frequenze
\end{tcolorbox} 

Essedo la funzione periodica si può scrivere la sequenza proprietà: 

{
    \Large 
    \begin{equation}
        e^{\jmath \left[ \left( 2 \pi - \omega_0\right) n + \phi\right]}
        = 
        e^{\jmath 2 \pi n} \cdot e^{\jmath \left( - \omega_0 n + \phi\right)}
    \end{equation}
}

\textbf{$3^{\circ}$ proprietà: } \newline 

\begin{tcolorbox}
    Questa è la proprietà più importante dell'esponeziale complesso !!!
\end{tcolorbox}

NON TUTTI gli esponenziali complessi sono periodici. \newline 

Deve esistere un N per cui: 

{
    \Large 
    \begin{equation}
        e^{\jmath \left[ \omega_0 \left( n + N\right) + \phi\right]}
        = 
        e^{\jmath \left(\omega_0 n + \phi\right)}
    \end{equation}
}

dove: 

{
    \Large 
    \begin{equation}
        \omega_0 N
        = 
        2 \pi k
    \end{equation}
}

Se N esiste, allora la sequenza esponenziale complesso puro sarà periodico nel periodo N. \newline 

\begin{tcolorbox}
    In altre parole, se vogliamo una sequenza esponenziale complessa periodica,  
    i valori di N e di k dipendono tutti da $\omega_0$ e devono rispettare l'uguaglianza scritta qui sopra
\end{tcolorbox}

\subsubsection{Sistema discreto lineare}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 11\\
    Appunti | 2025-09-30 | pag 1
}

Un operatore (o in questo caso sistema) $T \{ \}$ è lineare \textbf{se e solo se} vale il principio di sovrapposizione degli effetti. \newline 

In formule: 

{
    \Large 
    \begin{equation}
        \begin{cases}
        y_1 \left[ n \right]
        = 
        T \{ x_1 \left[ n\right]\}
        \\
        y_2 \left[ n \right]
        = 
        T \{ x_2 \left[ n\right]\}
        \end{cases}        
        \to 
        a \cdot y_1 \left[ n\right]
        + 
        b \cdot y_2 \left[ n\right]
        = 
        T 
        \{ 
        a \cdot y_1 \left[ n\right]
        + 
        b \cdot y_2 \left[ n\right]
        \}
    \end{equation}
}

dove a, b sono delle costanti. \newline 

In altri termini, se vale il principio di sovrapposizione degli effetti (che è una proprietà dei sistemi lineari), 
allora possiamo considerare l'uscita del sistema discreto lineare come combinazione lineare degli ingressi. \newline

\subsubsection{Sistema discreto stazionario}

\footnote{

    Appunti | 2025-09-30 | pag 1
}

Un operatore (o in questo caso sistema) $T \{ \}$ è stazionario (cioè a tempo-invariante) \textbf{se e solo se} 
la trasformazione che opera non dipende dall'istante di applicazione. \newline 

In formule: 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        T \{ x \left[ n \right]\}
        \\
        &\downarrow
        \\
        y \left[ n - n_0\right]
        &= 
        T \{ x \left[ n - n_0\right]\}
        \text{ }
        \forall n_0, x \left[ n\right]
        \end{split}
    \end{equation}
}

dove $n_0$ è un indice di ritardo e $x \left[ n\right]$ è una sequenza di ingresso all'operatore $T \{ \}$. \newline

\subsection{Sistemi discreti causali}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 13\\
    Appunti | 2025-09-30 | pag 1
}

\begin{tcolorbox}
La causalità è una proprietà molto importante dei sistemi discreti, 
perchè, in base alla sequenza di ingresso, 
l'andamento del sistema varierà di conseguenza  
\end{tcolorbox}

Un operatore (o in questo caso sistema) $T \{ \}$ è causale se 
la sua uscita all'istante $n_0$ dipende solo dagli ingressi con indice: 

{
    \Large 
    \begin{equation}
        k \leq n_0
    \end{equation}
}

In formule, se: 

{
    \Large 
    \begin{equation}
        \begin{cases}
            y_1 \left[ n\right]
            = 
            T \{ x_1 \left[ n\right]\}
            \\
            y_2 \left[ n\right]
            = 
            T \{ x_2 \left[ n\right]\}
        \end{cases}
    \end{equation}
}

allora: 

{
    \Large 
    \begin{equation}
        \begin{split}
            x_1 \left[ n\right] 
            &= 
            x_2 \left[ n\right]
            \\
            &\downarrow
            \\
            y_1 \left[ n\right] 
            &= 
            y_2 \left[ n\right] 
            \text{ }
            \forall n_0, x_1 \left[ n\right], x_2 \left[ n\right], n \leq n_0
        \end{split}
    \end{equation}
}

\begin{tcolorbox}[colback=red!5!white,colframe=red!75!black]
  L'indice k e l'indice n indicata nella formula sono gli stessi? (Penso di sì)
\end{tcolorbox}

\begin{tcolorbox}[colback=green!5!white,colframe=green!75!black]

\end{tcolorbox} 

Un esempio di un di sistema causale è la differenza in indietro: 

{
    \Large 
    \begin{equation}
        y \left[ n\right] 
        = 
        x \left[n\right]
        - 
        x \left[n - 1\right]
    \end{equation}
}

Invece, un sistema non causale è la differenza in avanti come il seguente: 

{
    \Large 
    \begin{equation}
        y \left[ n\right] 
        = 
        x \left[n + 1\right]

    \end{equation}

\subsection{Sistemi discreti stabili BIBO}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 14\\
    Appunti | 2025-09-30 | pag 1
}

L'operatore $T \{ \}$ si dice stabile BIBO (cioè Bounded Input Bounded Output) 
se, ad un ingresso \textbf{limitato}, corrisponde un'uscita \textbf{limitata}. \newline 

In formule, se: 

{
    \Large 
    \begin{equation}
        y \left[ n \right]
        = 
        T \{ x \left[ n\right]\}
    \end{equation}
}

allora: 

{
    \Large 
    \begin{equation}
        \begin{split}
            \abs{x \left[ n\right]} 
            \leq
            &B_x 
            < 
            + \infty
            \\
            &\downarrow
            \\
            \abs{y \left[ n\right]} 
            \leq
            &B_y 
            < 
            + \infty
            \text{ }
            \forall n, x \left[ n\right]
        \end{split}
    \end{equation}
}

dove $B_x$ e $B_y$ sono dei limiti superiori rispettivamente dell'ingresso e dell'uscita, 
i quali non tendono ad infinito. \newline 

\begin{tcolorbox}

Per adesso, l'opposto della stabilità, cioè l'instabilità del sistema, non è un problema. \newline 

Verrà approndito in seguito, perchè non è detto che sia un problema: 
come tutte le cose in ingegneria, dipende.
\end{tcolorbox}

\section{Risposta impulsiva in un sistema LTI}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 15\\
    Appunti | 2025-09-30 | pag 1
}

Per sistema LTI (cioè Linear Time-Invariant) si intende 
un sistema lineare a tempo invariante, cioè un sistema lineare che non varia le sue caratteristiche nel tempo. \newline 


Di seguito la definizione di risposta impulsiva. \newline 

Se $T \{ \}$ è un operatore LTI, 
allora è completamente caratterizzato dalla sua risposta all'\textbf{imuplso unitario}. \newline 

In formule, sia: 

{
    \Large 
    \begin{equation}
        h \left[ n \right] 
        = 
        T \{ \delta \left[ n \right] \}
    \end{equation}
}

dove $\delta \left[ n \right]$ è l'impulso unitario, 
allora possiamo scrivere che: 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right]
            &= 
            T \{ x \left[ n\right]\}
            \\
            &= 
            T 
            \left\{ 
                \sum_{k = - \infty}^{+ \infty}
                x \left[ k\right]
                \cdot 
                \delta \left[ n - k\right]
            \right\}
        \end{split}
    \end{equation}
}

Per linearità, 
posso portare l'operatore $T \{ \}$ dentro alla sommatoria, quindi: 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right]
            &= 
            T 
            \left\{ 
                \sum_{k = - \infty}^{+ \infty}
                x \left[ k\right]
                \cdot 
                \delta \left[ n - k\right]
            \right\}
            \\
            &=    
            \sum_{k = - \infty}^{+ \infty}
                x \left[ k\right]
                \cdot
                T 
            \left\{  
                \delta \left[ n - k\right]
            \right\}
            \\
            &= 
            \sum_{k = - \infty}^{+ \infty}
            x \left[ k\right]
            \cdot 
            h \left[ n - k \right]
        \end{split}
    \end{equation}
}

\begin{tcolorbox}

Possiamo svolgere questa semplificazione con $T \{ \}$ anche se non sappiamo nel dettaglio come è fatto l'operatore: 
l'unica cosa che sappiamo è la relazione tra $T \{ \}$ e l'ingresso $x \left[ n\right]$, 
che in questo caso è $\delta \left[ n\right]$, cioè l'impulso unitario.
    
\end{tcolorbox}

Quindi, possiamo esprimere l'uscita $y \left[ n\right]$ del sistema LTI, dato come ingresso un impulso unitario, 
con questa formula: 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        h \left[ n - k \right]
        \\
        &= 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ n - k\right]
        \cdot 
        h \left[ k \right]
        \end{split}
    \end{equation}
}

In questo caso, l'uscita $y \left[ n\right]$ prende il nome di prodotto di convoluzione. \newline 

\newpage 

\begin{tcolorbox}
Nel caso "analogico", la sommatoria sarebbe l'integrale di convoluzione, 
che nel corso di Teoria dei segnali con il mitico Chiaraluce, sarebbe: 

{
    \Large
    \begin{equation}
        \begin{split}
        y(t)
        &= 
        \int_{- \infty}^{+ \infty}
        f(\tau)
        \cdot 
        g(t - \tau) d\tau 
        \\
        &=
        \int_{- \infty}^{+ \infty}
        f(t - \tau)
        \cdot 
        g(t) d\tau 
        \end{split}
    \end{equation} 
}

Per approfondire nel mondo analogico: \\
\url{https://it.wikipedia.org/wiki/Convoluzione}
\end{tcolorbox}

Il prodotto di convoluzione significa che, sapendo che il sistema è LTI, cioè è indipendente dal tempo, 
basta solo sapere la risposta impulsiva $h \left[ n\right]$ per conoscere tutto il sistema $T \{ \}$. \newline 

Graficamente parlando, dato un sistema LTI: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Operatore T discreto.png}
\end{figure}  

e dato questo tipo di ingresso, cioè l'impulso unitario $\delta \left[ n \right]$: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.7]{Sequenza impulso unitario.png}
\end{figure}  

che poi deve essere ritardato per ogni indice di k, 
è possibile ricavare la risposta impulsiva del sistema LTI. \newline

\subsection{Prodotto di convoluzione e le sue proprietà}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 15 - 16\\
    Appunti | 2025-09-30 | pag 1
}

Il seguente prodotto di convoluzione: 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        h \left[ n - k \right]
        \\
        &= 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ n - k\right]
        \cdot 
        h \left[ k \right]
        \end{split}
    \end{equation}
}

può essere anche scritto con la seguente notazione: 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        x \left[ n\right]
        *
        h \left[ n\right]
        \\
        &= 
        h \left[ n\right]
        *
        x \left[ n\right]
        \end{split}
    \end{equation}
}

Di seguito le proprietà del prodotto di convoluzione. \newline 

\textbf{Proprietà commutativa} \newline 

Data una sequenza di ingresso $x \left[ n\right]$ e una risposta impulsiva $h \left[ n\right]$ di un sistema discreto LTI, 
allora possiamo applicare la proprietà commutativa nel prodotto di convoluzione, cioè: 

{
    \Large 
    \begin{equation}
        h \left[ n\right]
        *
        x \left[ n\right]
        =
        x \left[ n\right]
        *
        h \left[ n\right] 
    \end{equation}
}

\textbf{Proprietà distributiva} \newline 

Date due sequenze di ingresso $x_1 \left[ n\right]$, $x_2 \left[ n\right]$ e una risposta impulsiva $h \left[ n\right]$ di un sistema discreto LTI, 
allora possiamo applicare la proprietà distributiva nel prodotto di convoluzione, cioè: 

{
    \Large 
    \begin{equation}
        h \left[ n\right]
        *
        \left( 
            x_1 \left[ n\right] 
            +
            x_2 \left[ n\right]
        \right)
        = 
        h \left[ n\right]
        *
        x_1 \left[ n\right]
        +
        h \left[ n\right]
        *
        x_2 \left[ n\right]
    \end{equation}
} 

\textbf{Proprietà di linearità} \newline 

Date due sequenze di ingresso $x_1 \left[ n\right]$, $x_2 \left[ n\right]$ con i rispettivi coefficienti moltiplicativi a, b e una risposta impulsiva $h \left[ n\right]$ di un sistema discreto LTI, 
allora possiamo applicare la proprietà di linearità nel prodotto di convoluzione, cioè: 

{
    \Large 
    \begin{equation}
        h \left[ n\right]
        *
        \left( 
            a 
            \cdot
            x_1 \left[ n\right] 
            +
            b 
            \cdot
            x_2 \left[ n\right]
        \right)
        = 
        a 
        \cdot 
        \left( 
        h \left[ n\right]
        *
        x_1 \left[ n\right]
        \right)
        +
        b 
        \cdot 
        \left(
        h \left[ n\right]
        *
        x_2 \left[ n\right]
        \right)
    \end{equation}
}

\subsection{Come calcolare la risposta impulsiva in un sistema LTI}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 17\\
    Appunti | 2025-09-30 | pag 1
}

La risposta di un sistema può essere calcolata in due metodi. \newline 

Il primo metodo è attraverso la sovrapposizione delle risposte del sistema a ciascun impulso $x \left[ k\right]$, 
ciascuno con posizione k, cioè: 

{
    \Large 
    \begin{equation}
        y \left[ n\right]
        = 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        T \left\{ \delta \left[ n - k\right]\right\}
    \end{equation}
}

dove $T \left\{ \delta \left[ n - k\right]\right\}$ è la risposta del sistema discreto LTI all'impulso unitario ritardato dell'indice k. \newline 

\begin{tcolorbox}
    Il problema più grosso di questa equazione è la sommatoria con indice k che va da $- \infty$ a $+ \infty$, 
    cioè infinite sommatorie. \newline 

    Inoltre, per lo stesso problema, si dovrebbero fare infiniti ritardi del gradino unitario $\delta \left[ n\right]$. 
\end{tcolorbox}

Il secondo metodo è quello della somma dei prodotti fra i campioni $x \left[ n\right]$ e dei campioni della sequenza $h \left[ k\right]$ rovesciata e spostata di n, cioè: 

{
    \Large 
    \begin{equation}
        y \left[ n\right]
        = 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        h \left[ - \left( k - n\right)\right]
    \end{equation}
}

\begin{tcolorbox}[colback=red!5!white,colframe=red!75!black]
  $T \left\{ \delta \left[ n - k\right]\right\}$ e $h \left[ - \left( k - n\right)\right]$ intendono la stessa cosa, cioè la risposta impulsiva al sistema discreto LTI, 
  soltanto che uno è shiftato per l'indice k e uno per l'indice n, giusto?
\end{tcolorbox}

\begin{tcolorbox}
    Rispetto all'equazione precedente, questa volta il ritardo della risposta impulsiva $h \left[ - (k)\right]$ viene fatta per l'indice n e non per l'indice k. \newline 

    Inoltre, il ribaldamento (o rovesciamento) di $h \left[ (k - n)\right]$ viene fatta grazie al segno negativo davanti a (k - n). \newline 

    A lezione, per spiegare in modo grafico cosa è la convolutione, il prof ci ha fatto vedere questi due video: 

    \begin{itemize}
        \item \url{https://it.wikipedia.org/wiki/Convoluzione#/media/File:Convolution_of_box_signal_with_itself2.gif} 
        \item \url{https://it.wikipedia.org/wiki/Convoluzione#/media/File:Convolution_of_spiky_function_with_box2.gif}
    \end{itemize}

\end{tcolorbox}

\subsection{Risposta impulsiva in un sistema causale e stabile BIBO}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 18\\
    Appunti | 2025-09-30 | pag 2
}

Se il sistema è causale, cioè ha la seguente risposta impulsiva: 

{
    \Large 
    \begin{equation}
        h \left[ n\right] = 0 \text{ } \forall n < 0
    \end{equation}
}

allora, la risposta al sistema impulsivo $y [n]$ diventa: 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        T \left\{ \delta \left[ n - k\right]\right\}
        \\
        &=  
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        h \left[n - k\right]
        \\
        &= 
        \sum_{k = - \infty}^{+ \infty}
        h \left[ k\right]
        \cdot 
        x \left[n - k\right]
        \\
        &\downarrow
        \\
        y \left[ n\right]
        &= 
        \sum_{k = 0}^{+ \infty}
        h \left[ k\right]
        \cdot 
        x \left[n - k\right]
    \end{split}
    \end{equation}
}

\begin{tcolorbox}
    Tra il secondo ed il terzo passaggio si è svolta questa uguaglianza: 

    {
        \Large 
        \begin{equation}
        \sum_{k = - \infty}^{+ \infty}
        x \left[ k\right]
        \cdot 
        h \left[n - k\right]
        = 
        \sum_{k = - \infty}^{+ \infty}
        h \left[ k\right]
        \cdot 
        x \left[n - k\right]    
        \end{equation}
    }

    grazie alla proprietà commutativa del prodotto di convoluzione. 
\end{tcolorbox}

\begin{tcolorbox}[colback=red!5!white,colframe=red!75!black]
  I passaggi algebrici e le considerazioni sono giuste?
\end{tcolorbox}

Da un punto di vista grafico, possiamo visualizzare, come esempio, 
una risposta di un sistema causale $y \left[ n\right]$ come: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.2]{Sequenza causale di esempio.png}
\end{figure}  

Come possiamo notare dal grafico, per $k < 0$, la sequenza causale è nulla. \newline 

Inoltre, sapendo che nei sistemi causali: 

{
    \Large 
    \begin{equation}
        h \left[ n\right] = 0 \text{ } \forall n < 0
    \end{equation}
}

possiamo far diventare un sistema non causale a causale se aspettiamo sufficentemente indici in modo che si possa fare la convoluzione dall'indice $k = 0$. \newline 

Invece, per sistemi stabili BIBO (cioè Bounded Input Bounded Output), 
in cui: 

{
    \Large 
    \begin{equation}
        \begin{cases}
            \abs{x \left[ n\right]} 
            \leq
            B_x 
            < 
            + \infty
            \\
            \abs{y \left[ n\right]} 
            \leq
            B_y 
            < 
            + \infty
        \end{cases}
        \text{ }
        \forall n, x \left[ n\right]
    \end{equation}
}

allora, la risposta impulsiva in un sistema BIBO vale: 

{
    \Large 
    \begin{equation}
        \abs{y \left[ n\right]}
        \leq 
        \sum_{k = - \infty}^{+ \infty}
        \abs{h \left[ k\right]}
        \cdot 
        \abs{x \left[ n - k\right]}
        \leq 
        B_x 
        \cdot 
        \sum_{k = - \infty }^{+ \infty}
        \abs{h \left[ k\right]}
    \end{equation}
}

Affinchè ci sia stabilità per qualasiasi sequenza di ingresso $x \left[ n\right]$, 
basta che: 

{
    \Large 
    \begin{equation}
        \sum_{k = - \infty }^{+ \infty}
        \abs{h \left[ k\right]}
        < 
        + \infty
    \end{equation}
}

cioè, in altri termini, diremmo che la risposta impulsiva converge; 
questo ci permette di affermare che la stabilità del sistema LTI la si può osservare anche dalla sua risposta impulsiva. \newline 

Grazie alle osservazioni svolte per i sistemi causali ed i sistemi stabili BIBO, 
possiamo dire che la risposta impulsiva $h \left[ k\right]$ (o $h \left[ n\right]$ dipende sempre dall'indice di riferimento), 
caratterizza completamente un sistema LTI discreto. \newline

\section{Suddivisione dei sistemi LTI}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 19 - 20\\
    Appunti | 2025-09-30 | pag 2
}

I sistemi LTI si possono dividere in due tipi: 

\begin{itemize}
    \item FIR (cioè Finite Impulse Response) 
    \item IIR (cioè Infinite Impulse Response)
\end{itemize}

I sistemi FIR, come dice il nome, sono sistemi in cui la risposta impulsiva $h \left[ n\right]$ 
ha durata limitata e che quindi non si estende da $- \infty$ a $+ \infty$. \newline 

Se consideriamo la finestra compresa tra gli indici $n_1$ e $n_2$, 
possiamo visualizzare un esempio di risposta impulsiva $h \left[ n\right]$ FIR (con le note prese a lezione): 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.6]{Esempio risposta impulsiva FIR.png}
\end{figure}  

Possiamo affermare che, un sistema LTI è FIR se: 

{
    \Large 
    \begin{equation}
        h \left[ n\right]
        \neq  
        0 
        \text{ solo per }
        n_1 \leq n \leq n_2
    \end{equation}
}

Il sistemi LTI FIR saranno sempre stabile se: 

{
    \Large 
    \begin{equation}
        \abs{h \left[ n\right]} 
        < 
        + \infty
        \text{ }
        \forall 
        n
    \end{equation}
}

\newpage 

Invece i sistemi IIR sono sistemi in cui la risposta impulsiva ha durata illimitata (e quest'ultimo dettaglio può essere un pro ed un contro: lo vedremo meglio nei prossimi capitoli). \newline 

Quindi, rispetto al caso precedente dei FIR, la finestra non sarà limitata compresa tra due indici $n_1$ e $n_2$, 
bensì la finestra sarà tra $- \infty$ e $+ \infty$. \newline 

Un esempio di risposta impulsiva IIR: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.3]{Esempio riposta impulsiva IIR.png}
\end{figure}  

dove i puntini a destra e a sinistra della figura indicano che si estendono da $- \infty$ a $+ \infty$. \newline 

La stabilità del sistema LTI IIR sarà garantita solo se: 

{
    \Large 
    \begin{equation}
        \sum_{k = - \infty}^{+ \infty}
        \abs{h \left[ k\right]}
        <  
        + \infty
    \end{equation}
}

Quindi, per garantire la stabilità del sistema LTI non bisogna che la sua risposta impulsiva debba avere una finestra finita, 
l'importante che la sua risposta impulsiva converga ad un valore che non sia infinito. \newline 

\newpage 

\subsection{Esempio numerico di stabilità di un sistema IIR }

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 21\\
    Appunti | 2025-09-30 | pag 2
}

Ad esempio, data la formulazione matematica di una risposta impulsiva: 

{
    \Large 
    \begin{equation}
        h \left[ n\right]
        = 
        a^{n} \cdot u \left[ n\right]
    \end{equation}
}

dove $u \left[ n\right]$ è un impulso unitario e a è un coefficiente, 
possiamo affermare che, se: 

{
    \Large 
    \begin{equation}
        \begin{cases}
        h \left[ n\right]
        = 
        a^{n} \cdot u \left[ n\right]
        \\
        \abs{a} < 1
        \end{cases}
    \end{equation}
}

allora il sistema IIR è stabile perchè la risposta impulsiva converge. \newline 

Un esempio grafico di $h \left[ n\right]$ con $a = 0.5$: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.25]{Esempio IIR stabile.png}
\end{figure}  

Invece se: 

{
    \Large 
    \begin{equation}
        \begin{cases}
        h \left[ n\right]
        = 
        a^{n} \cdot u \left[ n\right]
        \\
        \abs{a} \geq 1
        \end{cases}
    \end{equation}
}

allora il sistema IIR è instabile perchè la risposta impulsiva non converge e tende a infinito. \newline 

Un esempio grafico di $h \left[ n\right]$ con $a = 1.5$: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.25]{Esempio FIR stabile.png}
\end{figure}

\subsection{Analisi di esempi di sistemi LTI }

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 22 - 23\\
    Appunti | 2025-09-30 | pag 2
}

Di seguito un elenco di sisemi discreti LTI, come si distinguono per la loro risposta impulsiva $h \left[ n\right]$e se sono FIR o IIR. \newline 

\textbf{Sistema identità} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right] &= x \left[ n \right]
            \\
            &\downarrow
            \\
            h \left[ n \right] &= \delta \left[ n\right]
        \end{split}
    \end{equation}
}

quindi, il sistema identià è un sistema FIR stabile, causale e stabile. \newline 

\textbf{Ritardo di M campioni} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right] &= x \left[ n - M\right]
            \\
            &\downarrow
            \\
            h \left[ n \right] &= \delta \left[ n - M\right]
        \end{split}
    \end{equation}
}

Il sistema è FIR e stabile, ma la causalità dipende dai valori di M. \newline 

Se $M > 0$, il sistema è causale. \newline  

Se $M < 0$, il sistema è non causala. \newline 

\textbf{Differenza in avanti} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right] &= x \left[ n + 1\right] - x \left[ n \right]
            \\
            &\downarrow
            \\
            h \left[ n \right] &= \delta \left[ n + 1\right] - \delta \left[ n \right]
        \end{split}
    \end{equation}
}

In questo caso, il sistema è FIR, non causale e stabile. \newline 

\textbf{Differenza all'indietro} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right] &= x \left[ n\right] - x \left[ n - 1 \right]
            \\
            &\downarrow
            \\
            h \left[ n \right] &= \delta \left[ n \right] - \delta \left[ n - 1\right]
        \end{split}
    \end{equation}
}

In questo caso, il sistema è FIR, causale e stabile. \newline 

\newpage 

\textbf{Accumulatore} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        \sum_{k = - \infty}^{n}
        x \left[ k\right]
       \\
       &\downarrow
       \\
       h \left[ n\right]
       &= 
        \sum_{k = - \infty}^{n} 
        \delta \left[ k\right]
        = 
        u \left[ n\right]
    \end{split}
    \end{equation}
}

dove $u \left[ n\right]$ è un gradino. \newline 

Il sistema è IIR, causale ma non stabile. \newline 

\textbf{Accumulatore ritardato} \newline 

{
    \Large 
    \begin{equation}
        \begin{split}
        y \left[ n\right]
        &= 
        \sum_{k = - \infty}^{n}
        x \left[ k - M\right]
       \\
       &\downarrow
       \\
       h \left[ n\right]
       &= 
        \sum_{k = - \infty}^{n} 
        \delta \left[ k - M \right]
        = 
        u \left[ n - M\right]
    \end{split}
    \end{equation}
}

dove $u \left[ n\right]$ è un gradino ed M è un indice di ritardo. \newline 

Il sistema è IIR, non stabile, ma la causalità dipende dal ritardo M. \newline 

Se $M > 0$, il sistema è causale (cioè è in ritardo). \newline  

Se $M < 0$, il sistema è non causala (cioè è in anticipo). \newline

\section{Connessioni dei sistemi LTI}

\footnote{ 
    Slide del prof | 1 - Segnali e Sistemi a Tempo Discreto | pag 24\\
    Appunti | 2025-09-30 | pag 2 
}

Come abbiamo visto ad eletttecnica, anche in DSP possiamo conntettere i "blocchetti". \newline 

Nel caso del DSP, non ci sono rami che sono bidirezionali, bensì unidirezionali. \newline 

Di seguito l'elenco delle connessioni di sistemi discreti LTI. \newline 

\textbf{Cascata / Serie} \newline

Dati due sistemi discreti LTI, possiamo collegarli in cascata (o serie) come in figura: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.2]{Sistemi LTI in cascata.png}
\end{figure}  

in cui: 

\begin{itemize}
    \item $h_1 \left[ n\right]$ e $h_2 \left[ n\right]$ sono risposte impulsive dei due sistemi discreti LTI 
    \item $x \left[ n\right]$ è la seguenza di ingresso 
    \item $z \left[ n\right]$ è la risposta in uscita al primo sistema LTI 
    \item $y \left[n \right]$ è la risposta in uscita da tutti e due i sistemi LTI
\end{itemize}

Allora possiamo scrivere le seguenti relazioni: 

{
    \Large 
    \begin{equation}
        \begin{cases}
            z \left[ n\right] = h_1 \left[ n\right] * x \left[ n\right] 
            \\
            y \left[ n\right] = h_2 \left[ n\right] * z \left[ n\right] 
        \end{cases}
        \to 
        y \left[ n\right]
        = 
        \left( 
            h_1 \left[ n\right]
            * 
            h_2 \left[ n\right]
        \right)
        * 
        x \left[ n\right]
    \end{equation}
}

L'espressione della sequenza in uscita $y \left[ n\right]$ esprime che 
$y \left[ n\right]$ è combinazione lineare delle risposte impulsive $h_1 \left[ n\right]$ e $h_2 \left[n\right]$: 
per questo motivo, volendo, possiamo spostare l'ordine dei due sistemi e l'uscita sarebbe la stessa. \newline 

Questo "scambio" non si può svolgere in sistemi discreti non lineari, 
in cui l'ordine dei blocchi è fondamentale. \newline 

\textbf{Parallelo} \newline

Dati due sistemi discreti LTI, possiamo collegarli in parallelo come in figura: 

\begin{figure}[h]
    \centering
    \includegraphics[scale = 0.2]{Sistemi LTI in parallelo.png}
\end{figure}  

Scrivendo le relazioni matematiche: 

{
    \Large 
    \begin{equation}
        \begin{split}
            y \left[ n\right]
            &= 
            x \left[  n \right] * h_1 \left[ n\right]
            +
            x \left[  n \right] * h_2 \left[ n\right] 
            \\
            &= 
            \left( h_1 \left[ n\right] + h_2 \left[ n\right]\right)
            *
            x \left[  n \right]
        \end{split}
    \end{equation}
}
